# 用户体验设计技能

## 📋 技能说明

设计优秀的用户体验，包括交互设计、信息架构和用户旅程。

---

## 🎯 适用场景

- 交互设计
- 用户流程设计
- 信息架构
- 可用性优化

---

## 📝 用户旅程图

```markdown
## 用户旅程：购买商品

### 阶段1：发现
- 触点：搜索/推荐
- 行为：浏览商品列表
- 情绪：好奇、期待
- 痛点：商品太多难以选择

### 阶段2：评估
- 触点：商品详情页
- 行为：查看详情、对比
- 情绪：犹豫、比较
- 痛点：信息不够详细

### 阶段3：购买
- 触点：购物车/结算
- 行为：下单支付
- 情绪：确定、期待
- 痛点：流程太长

### 阶段4：使用
- 触点：订单跟踪
- 行为：等待收货
- 情绪：期待
- 痛点：等待焦虑
```

---

## 📝 交互设计原则

1. **反馈及时** - 操作有明确反馈
2. **可撤销** - 错误可恢复
3. **一致性** - 相同操作相同结果
4. **防错设计** - 预防错误发生
5. **减少记忆负担** - 信息可见
