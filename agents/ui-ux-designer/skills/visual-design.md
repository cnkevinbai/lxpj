# 视觉设计技能

## 📋 技能说明

进行专业的视觉设计，包括色彩、排版、布局和图标设计。

---

## 🎯 适用场景

- 界面设计
- 视觉规范
- 图标设计
- 品牌视觉

---

## 📝 排版规范

```css
/* 字体家族 */
--font-sans: 'Inter', -apple-system, sans-serif;
--font-mono: 'JetBrains Mono', monospace;

/* 字号系统 */
--text-xs: 12px;
--text-sm: 14px;
--text-base: 16px;
--text-lg: 18px;
--text-xl: 20px;
--text-2xl: 24px;
--text-3xl: 30px;

/* 行高 */
--leading-tight: 1.25;
--leading-normal: 1.5;
--leading-relaxed: 1.75;

/* 字重 */
--font-normal: 400;
--font-medium: 500;
--font-semibold: 600;
--font-bold: 700;
```

---

## 📝 间距系统

/* 基于 4px 网格 */
