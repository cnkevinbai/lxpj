# 原型设计技能

## 📋 技能说明

创建交互原型和设计稿，用于验证设计和开发交付。

---

## 🎯 适用场景

- 低保真原型
- 高保真设计稿
- 交互原型
- 设计交付

---

## 📝 原型层级

```
低保真原型 → 中保真原型 → 高保真原型
    │              │              │
   结构          内容+布局       视觉+交互
```

---

## 📝 设计交付清单

```markdown
## 设计交付文档

### 1. 设计稿
- [ ] 所有页面设计稿
- [ ] 响应式版本
- [ ] 状态变体

### 2. 标注
- [ ] 尺寸标注
- [ ] 颜色标注
- [ ] 字体标注
- [ ] 间距标注

### 3. 资源
- [ ] 切图资源
- [ ] 图标 SVG
- [ ] 字体文件

### 4. 说明文档
- [ ] 设计说明
- [ ] 交互说明
- [ ] 动效说明
```
