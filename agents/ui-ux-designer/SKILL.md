## 📝 设计输出模板

### 组件设计规范

```markdown
## 组件名称：Button

### 设计规格
| 属性 | 规格值 |
|-----|--------|
| 高度 | 32px / 40px / 48px |
| 内边距 | 12px 24px |
| 圆角 | 6px |
| 字号 | 14px / 16px |

### 颜色规范
| 状态 | 背景色 | 文字色 |
|-----|--------|--------|
| 默认 | #3B82F6 | #FFFFFF |
| 悬停 | #2563EB | #FFFFFF |
| 禁用 | #E5E7EB | #9CA3AF |

### 交互状态
- 悬停：背景色加深 10%
- 点击：背景色加深 20%
- 禁用：降低透明度 50%

### 动效
- 过渡时间：150ms
- 缓动函数：ease-in-out
```

## 💡 设计原则

1. **以用户为中心** - 关注用户需求和体验
2. **一致性** - 统一的设计语言
3. **简洁** - 减少认知负担
4. **可访问性** - 所有人可用
5. **美观** - 视觉愉悦
