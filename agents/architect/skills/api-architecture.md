# API 架构设计技能

## 📋 技能说明

设计 API 网关、服务拆分、版本控制等 API 层面的架构。

---

## 🎯 适用场景

- API 网关设计
- 服务拆分
- 接口版本管理
- 契约设计

---

```
┌─────────────────────────────────────────────┐
│                  API Gateway                 │
├─────────────────────────────────────────────┤
│  ┌────────────────────────────────────────┐ │
│  │         功能层                          │ │
│  │  认证 │ 限流 │ 路由 │ 熔断 │ 监控       │ │

│  │         服务层                          │ │

└─────────────────────────────────────────────┘
```
