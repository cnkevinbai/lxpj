# 需求分析技能

## 📋 技能说明

分析用户需求，转化为可执行的产品需求文档和用户故事。

---

## 🎯 适用场景

- 需求收集
- 需求分析
- 用户故事编写
- 验收标准定义

---

## 📝 用户故事模板

```markdown
## 用户故事：[功能名称]

### 描述
作为 [角色]
我希望 [行为]
以便 [价值]

### 验收标准
- [ ] Given [前置条件] When [操作] Then [预期结果]
- [ ] Given [前置条件] When [操作] Then [预期结果]

### 优先级
P0 / P1 / P2 / P3

### 工作量估算
S / M / L / XL

### 相关模块
- [ ] 前端
- [ ] 后端
- [ ] 数据库
```

---

## ✅ 检查清单

- [ ] 用户故事符合 INVEST 原则
- [ ] 验收标准可测试
- [ ] 优先级合理
- [ ] 有清晰的边界
