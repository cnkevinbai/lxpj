# 安全审计技能

## 📋 技能说明

对代码和系统进行安全审计，发现和修复安全漏洞。

---

## 🎯 适用场景

- 代码安全审计
- 配置安全检查
- 漏洞扫描
- 权限审计

---

## 📝 OWASP Top 10 检查

| 漏洞 | 检查项 | 修复建议 |
|-----|--------|---------|
| SQL注入 | 参数化查询 | 使用 ORM 或预处理语句 |
| XSS | 输出编码 | 转义特殊字符 |
| CSRF | Token 验证 | 使用 CSRF Token |
| 认证失效 | 密码策略 | 强密码 + 多因素认证 |
| 访问控制失效 | 权限检查 | 最小权限原则 |

---

## ✅ 检查清单

- [ ] 输入验证完整
- [ ] 输出编码正确
- [ ] 认证机制安全
- [ ] 权限控制完善
- [ ] 敏感数据加密
