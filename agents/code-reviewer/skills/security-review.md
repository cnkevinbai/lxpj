# 安全审查技能

## 📋 技能说明

从安全角度审查代码，发现安全漏洞。

---

## 🎯 适用场景

- 代码安全审查
- 漏洞发现
- 安全加固

---

## 📝 检查项

| 类别 | 检查内容 |
|-----|---------|
| 输入验证 | XSS、SQL注入、命令注入 |
| 认证授权 | 越权访问、会话管理 |
| 敏感数据 | 加密存储、传输安全 |
| 错误处理 | 信息泄露、异常暴露 |

---

## ✅ 检查清单

- [ ] 所有输入已验证
- [ ] 敏感数据已加密
- [ ] 权限检查完整
- [ ] 日志不含敏感信息
