# 代码质量审查技能

## 📋 技能说明

审查代码质量，发现代码异味和改进点。

---

## 🎯 适用场景

- 代码审查
- 重构建议
- 代码规范检查

---

## 📝 审查报告模板

```markdown
## 代码审查报告

### 📊 总体评估

| 维度 | 评分 | 说明 |
|-----|------|------|
| 可读性 | ⭐⭐⭐⭐ | 命名清晰 |
| 可维护性 | ⭐⭐⭐ | 部分函数过长 |
| 性能 | ⭐⭐⭐⭐ | 有缓存机制 |

### 🔴 必须修复

1. **问题** - 文件:行号
   ```diff
   - 原代码
   + 建议修改
   ```

### 🟡 建议改进

1. **问题** - 文件:行号
   - 建议：...
```

---

## ✅ 检查清单

- [ ] 命名规范
- [ ] 代码结构
- [ ] 注释完整
- [ ] 无重复代码
