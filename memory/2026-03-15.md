## 🎯 重要事件

### memU-engine 记忆引擎安装 ✅

**时间**: 13:15
**版本**: 0.3.4
**作者**: duxiaoxiong

**安装原因**:
- 主人指出我每天凌晨重置，没有记性
- 需要长期记忆系统来跨会话保持记忆
- memU 比手动 MEMORY.md 更强大

**核心特性**:
- ✅ SQLite + 向量数据库存储
- ✅ 自动从对话提取记忆（档案、事件、知识、技能）
- ✅ 语义搜索（非关键词匹配）
- ✅ 多代理隔离 + 显式共享
- ✅ 零冗余 Token 设计

**配置**:
```json
{
  "embed": {
    "model": "text-embedding-3-small",
    "dimensions": 1536
  },
  "llm": {
    "model": "qwen3.5-plus"
  },
  "language": "zh",
  "memoryRoot": "~/.openclaw/memUdata/memory",
  "search": {
    "mode": "fast"
  },
  "documentSync": {
    "includeDefaultPaths": true,
    "extraPaths": ["workspace/MEMORY.md", "workspace/memory/"]
  }
}
```

**存储结构**:
```
~/.openclaw/memUdata/
├── conversations/      # 会话片段
├── resources/          # 导入文档
├── memory/
│   ├── shared/memu.db  # 共享文档
│   └── main/memu.db    # 主代理记忆
└── state/              # 同步状态
```

**使用方式**:
- 直接说 "Call memory_search" 即可搜索记忆
- 后台同步服务自动启动
