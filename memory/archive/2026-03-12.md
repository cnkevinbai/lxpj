### 部署方案
```bash
# 方案一：传统部署
psql -U evcart -d evcart -f database/migrations/*.sql
cd backend && npm install && npm run build && npm run start:prod
cd crm && npm install && npm run build

# 方案二：Docker 部署
docker-compose up -d
```
