## 核心业务流程
```
销售流程：线索 → 商机 → 客户 → 合同 → 订单 → 交付 → 回款
ERP 流程：采购 → 生产 → 质检 → 入库 → 销售
服务流程：工单 → 派单 → 处理 → 完成 → 评价
财务流程：应收 → 应付 → 报销 → 核算 → 报表
```
