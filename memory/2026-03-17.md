## 道达智能官网项目

### 完成页面（8个）
1. 登录页 `/login` - 数字化门户登录 V2
2. 首页 `/` - 品牌官网首页 V2
3. 关于我们 `/about` - 企业门户
4. 产品中心 `/products` - 产品矩阵中心
5. CRM客户管理 `/portal/crm` - 客户管理表格
6. ERP库存管理 `/erp/dashboard` - 库存概览仪表盘
7. 成功案例中心 `/cases` - 案例展示
8. 行业智慧方案 `/solutions` - 方案架构

### 技术栈
- React 18 + TypeScript + Vite
- 纯 CSS（无 Tailwind）
- 中英双语切换
- 深色主题

### 关键修复
- **CSS 类名冲突**：全局 `index.css` 与页面样式冲突
- **解决方案**：所有类名添加 `daoda-` 前缀
- **公共样式**：创建 `styles/common.css` 统一设计系统

### 设计变量
```css
--daoda-primary: #6600ff
--daoda-bg: #0A0A0A
--daoda-header-height: 80px
--daoda-radius: 16px
```

### 公共组件
- `.daoda-header` - 固定导航栏
- `.daoda-footer` - 统一页脚
- `.daoda-btn-primary` - 主按钮
- `.daoda-card` - 卡片
- `.daoda-glass-panel` - 玻璃面板
- `.daoda-badge-*` - 状态徽章
- `.daoda-table` - 数据表格

### 服务管理
- 使用 `pm2` 持久运行 Vite 开发服务器
- 端口：3001
- 命令：`pm2 list | restart | logs`
