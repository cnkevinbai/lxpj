### 学习成果

#### 1. 动态规划模式掌握

**五大 DP 模式**:
| 模式 | 特征 | 代表问题 |
|------|------|----------|
| 线性 DP | 单向前进 | 最长递增子序列 |
| 区间 DP | 区间合并 | 矩阵链乘法 |
| 背包 DP | 选/不选 | 0/1 背包 |
| 树形 DP | 树上递归 | 树的最大独立集 |
| 状态压缩 DP | 状态用位表示 | 旅行商问题 |

**从暴力到 DP 思维路径**:
1. 暴力递归 → 发现重复计算
2. 识别状态 → 记忆化
3. 状态转移方程
4. 空间优化

#### 2. 复杂度直觉

**关键量级**:
- O(log n): 1000 → 10, 100万 → 20
- O(n log n): 100万 → 2000万次操作
- O(n²): 1万 → 1亿次操作 (开始慢了)

**数据结构选择直觉**:
- 快速查找 → Hash Table O(1)
- 有序遍历 → 平衡树 O(log n)
- 插入/删除头部 → 链表 O(1)
