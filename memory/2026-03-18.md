## DAOD 车联网管理平台 - 项目接管

### 项目概况

| 项目 | 内容 |
|------|------|
| 名称 | 车联网管理平台 (IoV Platform) |
| 所属 | 四川道达智能车辆制造有限公司 |
| 位置 | `/home/3844778_wy/.openclaw/workspace/projects/DAOD/` |
| 场景 | 无人驾驶电动观光车 + 新能源电动观光车 |

### 核心特性

- **模块化架构**: 所有功能模块独立开发、独立部署
- **热插拔热更新**: 支持模块动态加载/卸载/更新
- **云边端协同**: 云端中心 + 边缘节点 + 车载终端
- **多租户账号体系**: 生产厂家、经销商、运营者、景区管理员多角色
- **双链路通信**: MQTT主链路 + TCP备用链路

### 技术栈

| 层级 | 技术 |
|------|------|
| 微服务 | Spring Cloud Alibaba |
| 消息 | EMQX + Kafka |
| 数据库 | PostgreSQL + TimescaleDB + ClickHouse |
| 缓存 | Redis |
| 容器 | Docker + Kubernetes |
| 监控 | Prometheus + Grafana |
