## 💡 经验教训

1. **路由嵌套** - 模块路由需要正确配置 `<Routes>` 和子 `<Route>`
2. **Provider 顺序** - `QueryClientProvider` 必须包裹在组件树顶层
3. **路径命名** - API 路径（复数）与数据库模块代码（单数）需要映射
4. **JWT Payload** - 角色信息应该存储角色代码而非角色 ID
5. **类型转换** - Query string 参数需要 `enableImplicitConversion` 或 `@Type()` 装饰器

## 💡 今日经验教训

1. **Spring Boot 版本兼容** - Boot 3.x 需要 Java 17，Boot 2.7.x 支持 Java 11
2. **Stream API 兼容** - `toList()` 是 Java 16+ 特性，Java 11 需用 `collect(Collectors.toList())`
3. **Lombok boolean getter** - `boolean` 字段的 getter 是 `isXxx()` 而非 `getXxx()`
4. **Maven repackage** - 库模块需要跳过 spring-boot-maven-plugin:repackage
5. **ModuleException 构造器** - 需要 (code, moduleName, message, cause) 四参数形式
6. **ThreadPoolExecutor** - `getActiveCount()` 需要具体实现类，不能是 ExecutorService 接口
