## 📋 项目概述

- **风格**: 方案一 NEXUS - 沉浸式科技感

- **色调**: 深邃黑 + 道达蓝渐变
- **特点**: 全屏视频、丰富动效、极简导航

| **业务定位** | ❌ SaaS 软件公司 | ✅ 新能源车辆制造商 |

| **官网用途** | ❌ 软件产品展示 | ✅ 车辆产品展示 + 品牌 |

### Phase 2: 官网首页重构 (3-4 天)

#### 2.1 布局实现
```
优先级:
P0 - Hero 区域 (全屏视频轮播)
P1 - 产品展示区 (5 大产品卡片)
P2 - 技术优势区 (左右交替)
P3 - 全球案例区 (地图交互)
P4 - 页脚
```

#### 2.2 动效实现
```css
重点动效:
1. Hero 文字渐变显现 (3s)
2. 粒子背景效果 (Canvas)
3. 卡片悬停三部曲 (上浮 + 光晕 + 阴影)
4. 滚动视差 (背景/前景不同速)
5. 数字滚动计数 (2s)
6. 按钮涟漪反馈 (0.6s)
```

#### 2.3 响应式适配
```
断点:
- Mobile: < 768px (单列)
- Tablet: 768px - 1024px (双列)
- Desktop: > 1024px (三列/四列)
```

## ⚠️ 风险点

### 技术风险
1. **动效性能** - 复杂动效可能影响性能
   - 缓解：性能测试 + 降级方案
2. **视频加载** - 全屏视频可能加载慢
   - 缓解：CDN + 懒加载 + 压缩
3. **浏览器兼容** - 部分特效不支持旧浏览器
   - 缓解：特性检测 + 降级方案

### 内容风险
1. **素材缺失** - 产品图/视频未及时提供
   - 缓解：提前沟通 + 占位图
2. **文案变更** - 产品参数频繁修改
   - 缓解：CMS 管理 + 版本控制

### 时间风险
1. **需求变更** - 设计过程中调整方向
   - 缓解：确认设计稿后再开发
2. **审批流程** - 设计稿审批耗时
   - 缓解：并行工作 + 提前沟通
