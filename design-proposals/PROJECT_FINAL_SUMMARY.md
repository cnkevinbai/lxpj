## 📊 项目概况

### 项目定位
为四川道达智能车辆制造有限公司打造**企业官网 + 内部数字化平台**，实现：
- 企业品牌形象展示
- 产品与服务介绍
- 内部业务管理系统
- 统一数字化工作平台

### 技术栈
- **前端框架**: React 18 + Vite 5
- **UI 组件库**: Ant Design 5
- **路由管理**: React Router v6
- **国际化**: i18next
- **状态管理**: React Hooks
- **构建工具**: Vite
