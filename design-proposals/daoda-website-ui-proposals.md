# 四川道达智能车辆制造有限公司 - 官网 UI 设计方案

## 项目概述

**客户**：四川道达智能车辆制造有限公司  
**业务范围**：新能源观光车、电动巡逻车、高尔夫球车、无人驾驶观光车、景区共享漫游车  
**目标市场**：国内 + 国际出口  
**设计风格**：参考大疆创新 (DJI) 官网风格 - 极简主义、科技感、现代感

---

# 方案一：「极境 · 未来」(NEXUS) - 终极版

## 1. 设计理念说明

「极境·未来」方案以"科技无界，出行无限"为核心理念，打造沉浸式的产品体验空间。设计灵感来源于未来交通的无限可能，通过极致的视觉张力和流畅的交互体验，展现道达智能车辆在新能源领域的技术实力。整体采用深色主题营造高端科技感，配合动态光影效果，让每一款产品都如同艺术品般呈现。设计强调"少即是多"的哲学，用大量留白突出产品本身，让用户在浏览过程中自然感受到品牌的创新精神和国际化视野。

**终极版升级**:
- ✅ 新增「四川道达智能数字化平台」统一登录入口
- ✅ 内外网双门户架构，统一品牌视觉
- ✅ **30+ 种酷炫动效**，打造沉浸式体验
- ✅ **黑蓝科技配色**，未来感十足
- ✅ **响应式导航**：PC 横向 / 移动汉堡
- ✅ **标题精炼**：4-12 字
- ✅ **浏览器标题**：道达智能车辆制造创新平台

## 2. 色彩规范 - 黑蓝科技风

| 类型 | 色值 | 用途 | 说明 |
|------|------|------|------|
| **主色** | `#050505` (极邃黑) | 主背景 | 接近纯黑，更深邃 |
| **主色** | `#0A0A0A` (深邃黑) | 次级背景 | 板块区分 |
| **主色** | `#121212` (碳素黑) | 卡片底色 | 悬浮元素 |
| **主色** | `#1A1A1A` (深空黑) | 导航栏 | 半透明 |
| **辅色** | `#FFFFFF` (纯白) | 主标题 | 100% 不透明 |
| **辅色** | `#E0E0E0` (浅灰白) | 正文 | 87% 不透明 |
| **辅色** | `#A0A0A0` (中灰) | 次要文字 | 60% 不透明 |
| **辅色** | `#606060` (深灰) | 边框/分割线 | 38% 不透明 |
| **强调色** | `#0066FF` (道达蓝) | 主强调色 | CTA、链接、高亮 |
| **强调色** | `#0088FF` (亮蓝) | 悬停态 | Hover 效果 |
| **强调色** | `#00AAFF` (天蓝) | 渐变起点 | 光效 |
| **强调色** | `#00D4FF` (科技蓝) | 渐变终点 | 动效光晕 |
| **强调色** | `#00FF88` (能量绿) | 新能源标识 | 成功状态 |
| **强调色** | `#FF3366` (警示红) | 警告/错误 | 少量使用 |

**渐变方案**：
```css
/* 主渐变 - 蓝调科技 */
--gradient-primary: linear-gradient(135deg, #0066FF 0%, #00D4FF 100%);

/* 光晕效果 - 呼吸感 */
--glow-blue: radial-gradient(circle, rgba(0,102,255,0.2) 0%, transparent 70%);

/* 能量线 - 流动感 */
--energy-line: linear-gradient(90deg, 
  transparent 0%, 
  #0066FF 25%, 
  #00D4FF 50%, 
  #0066FF 75%, 
  transparent 100%
);

/* 金属质感 - 科技感 */
--metallic: linear-gradient(180deg, 
  #1A1A1A 0%, 
  #252525 50%, 
  #1A1A1A 100%
);
```

**色彩使用原则**：
- 背景：90% 黑色系 (#050505 - #1A1A1A)
- 文字：10% 白色系 (#FFFFFF - #606060)
- 强调：<5% 蓝色系 (#0066FF - #00D4FF)
- 产品：唯一的主要色彩来源

```javascript
// 数字从 0 滚动到目标值
function animateNumber(element, target, duration = 2000) {
  const start = 0;
  const startTime = performance.now();
  
  return function update(currentTime) {
    const elapsed = currentTime - startTime;
    const progress = Math.min(elapsed / duration, 1);
    
    // easing: easeOutExpo
    const ease = progress === 1 ? 1 : 1 - Math.pow(2, -10 * progress);
    
    const current = Math.floor(start + (target - start) * ease);
    element.textContent = current.toLocaleString();
    
    if (progress < 1) {
      requestAnimationFrame(update);
    }
  };
}
```

# 方案二：「纯净 · 至简」(PURE)

## 1. 设计理念说明

「纯净·至简」方案以"白色画布，产品主角"为设计哲学，打造极致简洁的视觉体验。整个网站如同一个现代艺术画廊，纯白背景让每一款智能车辆都成为焦点。设计灵感来源于苹果官网的极简美学，用最少的元素传达最清晰的信息。大量留白不仅营造呼吸感，更引导用户专注于产品本身。浅灰色调的层次变化创造微妙的空间感，而品牌蓝作为唯一强调色，精准引导用户注意力。这种克制的设计语言，恰恰体现了道达对产品的自信——不需要花哨的装饰，产品本身就是最好的证明。

# 方案三：「动能 · 突破」(DYNAMO)

## 1. 设计理念说明

「动能·突破」方案以"速度、力量、突破"为核心概念，打造充满动感的视觉体验。设计灵感来源于赛车运动的激情与工业设计的精密，通过大胆的视觉对比和动态元素，展现道达车辆的强劲性能。深色背景配合高对比度的产品特写，营造强烈的视觉冲击力。斜线、箭头等动态元素贯穿全站，暗示速度与前进。品牌蓝以渐变光效呈现，如同能量流动。这种充满张力的设计语言，精准传达道达在智能车辆领域的突破精神和技术实力，适合强调产品性能和国际市场竞争力的品牌定位。

## 2. 色彩规范

| 类型 | 色值 | 用途 |
|------|------|------|
| **主色** | `#0D0D0D` (极深黑) | 主背景 |
| **主色** | `#141414` (深黑) | 次级背景 |
| **主色** | `#1F1F1F` (炭黑) | 卡片背景 |
