## 📱 模块概述

AI 智能客服是基于大语言模型的企业级智能客服系统，提供 24/7 在线咨询服务，支持产品咨询、技术支持、售后问题等多种场景。

---

## 🎯 功能特性

### 核心功能 ✅
1. **智能对话** - 基于 LLM 的自然语言对话
2. **知识库问答** - 企业知识库智能检索
3. **多轮对话** - 上下文理解与记忆
4. **问题分类** - 自动识别问题类型
5. **人工转接** - 复杂问题转人工客服
6. **对话记录** - 完整对话历史保存

### 酷炫元素 ✨
1. **打字机效果** - 文字逐字显示动画
2. **语音输入** - 语音识别转文字
3. **快捷回复** - 常见问题快捷按钮
4. **情感识别** - 识别用户情绪
5. **个性化回复** - 根据用户画像定制回复
6. **多语言支持** - 中/英自动识别

## 🔧 技术栈

### 前端技术
- **框架**: React 18 + TypeScript
- **UI 库**: Ant Design 5
- **动画**: Framer Motion
- **语音**: Web Speech API
- **状态**: Zustand

### 后端技术
- **框架**: NestJS 10
- **AI 接口**: 通义千问/文心一言/OpenAI
- **数据库**: PostgreSQL 14
- **向量库**: pgvector / Pinecone
- **缓存**: Redis

### AI 模型
- **对话模型**: 通义千问 Max / GPT-4
- **嵌入模型**: text-embedding-3-large
- **知识库**: 企业产品/技术文档

## 📁 数据库设计

### 对话记录表
```sql
CREATE TABLE chat_conversations (
    id UUID PRIMARY KEY,
    user_id UUID,
    session_id VARCHAR(100),
    created_at TIMESTAMP,
    updated_at TIMESTAMP,
    status VARCHAR(20)
);

CREATE TABLE chat_messages (
    id UUID PRIMARY KEY,
    conversation_id UUID,
    role VARCHAR(20),  -- user/assistant/system
    content TEXT,
    tokens INT,
    created_at TIMESTAMP
);
```

### 知识库表
```sql
CREATE TABLE knowledge_base (
    id UUID PRIMARY KEY,
    category VARCHAR(50),
    title VARCHAR(200),
    content TEXT,
    embedding VECTOR(1536),
    created_at TIMESTAMP
);
```

## 🎨 UI 设计规范

位置：右下角

图标：客服头像/消息图标
动画：呼吸灯效果/轻微浮动

### 聊天窗口
```
尺寸：380x600px（桌面）
      100%x100%（手机）
背景：白色 #FFFFFF
圆角：16px
阴影：0 8px 32px rgba(0,0,0,0.12)

```
用户消息：

- 背景：浅灰 #F5F5F5

```
打字机效果：0.05s/字
消息出现：fade-in + slide-up 300ms
悬浮按钮：breath 2s infinite
```

## 📊 对话流程

```
用户进入
    │
    ▼
AI 主动问候
    │
    ▼
用户提问
    │
    ▼
AI 理解意图 ──▶ 知识库检索 ──▶ 生成回复
    │
    ▼
用户满意？───否──▶ 转人工客服
    │
   是
    │
    ▼

```
