## 📱 官网概述

四川道达智能车辆制造有限公司官方网站采用**大疆（DJI）官网设计风格**，基于 Next.js 14 开发，以极简设计、大幅产品图片、流畅动效为特色，展示公司形象和电动车产品。

---

## 🎨 大疆风格设计原则

### 1. 极简主义
- 大量留白
- 简洁的导航
- 清晰的层次结构

### 2. 产品为主
- 全屏产品图片展示
- 产品为视觉焦点
- 减少文字干扰

### 3. 色彩规范
- 主色：纯黑（#000000）+ 纯白（#FFFFFF）
- 辅助色：高级灰（#F5F5F5, #E0E0E0, #999999）
- 强调色：品牌蓝（#0070F3）或产品色

### 4. 字体规范
- 标题：粗体大字（32px-64px）
- 正文：中等字号（16px-18px）
- 辅助文字：小字灰色（14px, #999）

### 5. 布局特点
- 全屏 Hero 区域
- 卡片式产品展示
- 滚动视差效果
- 流畅过渡动效

## 🚀 开发指南

### 环境搭建
```bash
# 1. 安装依赖
cd website
npm install

# 2. 启动开发服务器
npm run dev

# 3. 访问
http://localhost:3000
```

```bash
# 构建生产版本
npm run build

npm start
```

### 部署
```bash
# Vercel 部署
vercel deploy

# Nginx 部署
npm run build
# 将 out/ 目录部署到 Nginx
```

## 📸 大疆风格参考

### 参考网站
- **大疆官网**: https://www.dji.com/cn
- **苹果官网**: https://www.apple.com
- **特斯拉官网**: https://www.tesla.com

### 设计要点
1. **产品图片质量第一** - 使用高质量产品摄影
2. **留白是王道** - 不要害怕大量留白
3. **文字要少** - 只保留必要信息
4. **字体要大** - 标题字号 40px+
5. **配色要简** - 黑白灰为主
6. **动效要流畅** - 滚动视差/淡入淡出

### 避免事项
- ❌ 花哨的背景
- ❌ 过多的颜色
- ❌ 密集的文字
- ❌ 复杂的布局
- ❌ 廉价的图片
