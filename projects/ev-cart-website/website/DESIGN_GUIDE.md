## 📱 官网概述

四川道达智能车辆制造有限公司官方网站是基于 Next.js 14 开发的企业官网，展示公司形象、产品信息、企业文化等内容，提供在线咨询和留言功能。

## 📱 页面设计

### 首页（page.tsx）✅

#### 布局结构
```
┌─────────────────────────────────┐
│  导航栏（Logo + 菜单 + 登录）   │
├─────────────────────────────────┤
│                                 │
│         Hero 区域               │
│   公司名 + 描述 + CTA 按钮       │
│                                 │
├─────────────────────────────────┤
│  核心特性（4 大系统卡片）        │
├─────────────────────────────────┤
│  项目统计（4 个统计项）          │
├─────────────────────────────────┤
│  页脚（版权 + 联系方式）         │
```

#### 组件说明
- **导航栏**: 固定顶部，包含 Logo、导航菜单、登录按钮

- **特性卡片**: 4 列布局，图标 + 标题 + 描述
- **统计卡片**: 4 列布局，数字 + 标签
- **页脚**: 深色背景，版权信息，联系方式

## 🔧 技术栈

### 前端框架
- **Next.js 14** - React 框架

- **Ant Design 5** - 组件库

## 🚀 开发指南

### 环境搭建
```bash
# 1. 安装依赖
cd website
npm install

# 2. 启动开发服务器
npm run dev

# 3. 访问
http://localhost:3000
```

### 构建发布
```bash
# 构建生产版本
npm run build

# 启动生产服务器
npm start
```

### 部署
```bash
# Vercel 部署
vercel deploy

# Nginx 部署
npm run build
# 将 out/ 目录部署到 Nginx
```
