## 📐 布局规范

### 栅格系统
```
最大宽度：1440px（桌面）
栅格列数：12 列
水槽宽度：80px
```

### 响应式断点
```
手机超小  320px   - 480px
手机      481px   - 767px
平板      768px   - 1023px
桌面      1024px  - 1439px
大屏      1440px+
```

### 布局原则
1. **大量留白** - 120-200px 区块间距
2. **居中对齐** - 内容居中，两侧留白
3. **视觉层次** - 字号/字重/颜色区分
4. **简洁导航** - 固定顶部，半透明
5. **全屏展示** - Hero 区域 100vh

## 🎬 动效规范

### 过渡动画
```
时长：300ms（标准）/ 500ms（慢）
缓动：ease-out / cubic-bezier(0.4, 0, 0.2, 1)
```

### 滚动动效
```
视差滚动：背景与内容不同步
淡入效果：内容滚动时渐显
缩放效果：卡片悬浮时放大
```

### 交互反馈
```
按钮悬浮：背景色变化 + 轻微放大
链接悬浮：下划线/颜色变化
卡片悬浮：阴影加深 + 轻微上浮
```

## 🔄 官网与 CRM 数据对接

### 数据流向

| 方向 | 数据类型 | 说明 |
|-----|---------|------|
| 官网→CRM | 客户注册 | 创建 CRM 客户 |
| 官网→CRM | 咨询表单 | 创建 CRM 线索 |
| 官网→CRM | 产品询价 | 创建 CRM 商机 |
| 官网→CRM | 服务申请 | 创建 CRM 工单 |
| CRM→官网 | 产品信息 | 产品列表/详情 |
| CRM→官网 | 新闻内容 | 新闻列表/详情 |
| CRM→官网 | 案例内容 | 案例列表/详情 |
| CRM→官网 | 统计数据 | 客户数/订单数 |

### API 接口

```typescript
// 客户相关
POST /api/v1/website/customers      // 客户注册
POST /api/v1/website/leads          // 线索创建
POST /api/v1/website/opportunities  // 商机创建
POST /api/v1/website/tickets        // 工单创建

// 产品相关
GET  /api/v1/website/products       // 产品列表
GET  /api/v1/website/products/:id   // 产品详情

// 内容相关
GET  /api/v1/website/news           // 新闻列表
GET  /api/v1/website/cases          // 案例列表
```
