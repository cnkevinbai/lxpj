### 1. 工单管理 ✅

**实体**: `ServiceTicket`

**工单类型**:
| 类型 | 代码 | 说明 |
|-----|------|------|
| 安装工单 | installation | 产品安装服务 |
| 维修工单 | repair | 产品维修服务 |
| 保养工单 | maintenance | 定期保养服务 |

**工单状态**:
```
待受理 → 已受理 → 已分配 → 处理中 → 待确认 → 已完成 → 已关闭

```

|-------|--------|------|
| 基础信息 | 8 | 工单号/类型/状态/优先级 |
| 客户信息 | 5 | 客户/电话/地址 |
| 产品信息 | 3 | 订单/产品/名称 |
| 问题描述 | 3 | 描述/图片/预约时间 |
| 服务人员 | 3 | 技师/方案/备件 |
| 费用信息 | 4 | 服务费/备件费/保修 |
| 评价信息 | 3 | 满意度/评价 |
| 时间信息 | 5 | 创建/受理/分配/完成/关闭 |

## 📊 业务流程

### 完整流程

```
客户发起服务请求
    ↓
创建服务工单 (pending)
    ↓
客服受理 (accepted)
    ↓
分配服务人员 (assigned)
    ↓
上门服务 (processing)
    ↓
服务完成 (waiting_confirm)
    ↓
客户确认 + 评价 (closed)
    ↓
工单归档
```

### 状态流转

```typescript
// 创建工单
pending → accepted (受理)

// 分配技师
accepted → assigned (分配)

// 开始服务
assigned → processing (处理中)

// 服务完成
processing → waiting_confirm (待确认)

// 客户确认
waiting_confirm → closed (已关闭)

// 取消工单
any → cancelled (已取消)
```
