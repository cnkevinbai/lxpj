## 🔄 业务流程

### 完整流程

```
客户申请服务 (APP/电话/ Web)
    ↓
创建服务工单 (pending)
    ↓
客服受理 (accepted)
    ↓
分配服务网点/人员 (assigned)
    ↓
上门服务 (processing)
    ↓
填写服务记录
    ↓
客户确认 + 评价 (closed)
    ↓
服务记录回流 CRM
    ↓
完善客户画像
```

### 状态流转

```typescript
// 正常流程
pending → accepted → assigned → processing → waiting_confirm → closed

// 取消流程
any → cancelled

// 客户确认
waiting_confirm → closed (客户确认)
```
