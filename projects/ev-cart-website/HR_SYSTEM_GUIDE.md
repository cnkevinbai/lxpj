## 📊 系统概览

道达智能 HR 系统覆盖人力资源管理全流程，从招聘到入职、从考勤到绩效、从培训到薪酬，实现人力资源一体化管理。

### 核心模块
| 模块 | 功能 | 状态 |
|------|------|------|
| 员工档案 | 员工信息、合同、档案 | ✅ |
| 招聘管理 | 职位、简历、面试、入职 | ✅ |
| 考勤管理 | 考勤记录、统计、请假 | ✅ |
| 绩效管理 | 绩效考核、评估、反馈 | ✅ |
| 培训管理 | 培训计划、执行、效果 | ✅ |
| 薪酬管理 | 薪资核算、发放、个税 | ⏳ |

## 🔄 核心业务流程

### 招聘入职流程
```
职位发布 → 简历筛选 → 面试 → Offer → 入职办理 → 试用期 → 转正
```

### 考勤流程
```
排班 → 打卡 → 考勤统计 → 请假/加班处理 → 月度报表 → 薪资核算
```

### 绩效流程
```
目标设定 → 过程跟踪 → 绩效评估 → 绩效反馈 → 结果应用
```

### 培训流程
```
需求调研 → 计划制定 → 培训执行 → 效果评估 → 档案记录
```

## 📈 关键指标 (KPI)

| 指标 | 公式 | 目标值 |
|------|------|--------|
| 招聘完成率 | 实际入职数 / 计划招聘数 | ≥ 90% |
| 招聘周期 | 职位发布到入职天数 | ≤ 30 天 |
| 考勤准确率 | 准确考勤记录 / 总记录 | ≥ 99% |
| 绩效考核覆盖率 | 参与考核人数 / 应考核人数 | 100% |
| 培训覆盖率 | 参加培训人数 / 总人数 | ≥ 80% |
| 员工流失率 | 离职人数 / 平均人数 | ≤ 10% |
| 员工满意度 | 满意员工数 / 总员工数 | ≥ 85% |

## 🔧 技术实现

### 数据表结构
```sql
-- 员工表
CREATE TABLE hr_employees (
  id UUID PRIMARY KEY,
  employee_number VARCHAR(50),
  name VARCHAR(100),
  gender VARCHAR(10),
  birthday DATE,
  phone VARCHAR(50),
  email VARCHAR(100),
  department_id UUID,
  position VARCHAR(100),
  hire_date DATE,
  status VARCHAR(20),
  created_at TIMESTAMP
);

-- 简历表
CREATE TABLE hr_resumes (
  id UUID PRIMARY KEY,
  job_id UUID,
  name VARCHAR(100),
  phone VARCHAR(50),
  email VARCHAR(100),
  education VARCHAR(50),
  experience TEXT,
  status VARCHAR(20),
  created_at TIMESTAMP
);

-- 考勤记录表
CREATE TABLE hr_attendance (
  id UUID PRIMARY KEY,
  employee_id UUID,
  date DATE,
  clock_in TIMESTAMP,
  clock_out TIMESTAMP,
  status VARCHAR(20),
  created_at TIMESTAMP
);

-- 绩效考核表
CREATE TABLE hr_performance (
  id UUID PRIMARY KEY,
  employee_id UUID,
  period VARCHAR(20),
  score DECIMAL(5,2),
  level VARCHAR(10),
  feedback TEXT,
  created_at TIMESTAMP
);

-- 培训记录表
CREATE TABLE hr_training_records (
  id UUID PRIMARY KEY,
  employee_id UUID,
  training_id UUID,
  status VARCHAR(20),
  score DECIMAL(5,2),
  feedback TEXT,
  completed_at TIMESTAMP
);
```
