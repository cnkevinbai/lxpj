## 📱 方案一：PWA (渐进式 Web 应用)

| 平台 | 支持度 | 说明 |
|-----|--------|------|
| Android | ✅ | Chrome 支持完美 |
| iOS | ✅ | Safari 11.3+ 支持 |
| HarmonyOS | ✅ | 浏览器支持 |

### 优点

- ✅ 无需应用商店

| 测试优化 | 2 天 | 多端测试 |
| **总计** | **6 天** | - |

## 🎯 推荐方案：Capacitor

### 选择理由

1. **技术栈匹配**: 基于 React，可复用现有代码
2. **原生支持**: 完善的原生 API 支持
3. **鸿蒙兼容**: 可通过 Web view 适配鸿蒙
4. **成本可控**: 20 天完成三端适配
5. **生态丰富**: Ionic 插件生态丰富
6. **维护活跃**: Ionic 团队持续维护

### 技术架构

```
┌─────────────────────────────────────────────────────────────┐
│                      应用层                                  │
│  ┌─────────────────┐  ┌─────────────────┐                  │
│  │   Android App   │  │     iOS App     │                  │
│  │   (Capacitor)   │  │   (Capacitor)   │                  │
│  └─────────────────┘  └─────────────────┘                  │
│  ┌─────────────────┐                                        │
│  │  HarmonyOS App  │                                        │
│  │  (Web View)     │                                        │
│  └─────────────────┘                                        │
├─────────────────────────────────────────────────────────────┤
│                      桥接层                                  │
│                    Capacitor JS                             │
├─────────────────────────────────────────────────────────────┤
│                      应用层                                  │
│                  React + TypeScript                         │
│                  (现有代码复用 90%+)                         │
└─────────────────────────────────────────────────────────────┘
```

### 实施步骤

#### 第一阶段：环境搭建 (2 天)

```bash
# 安装 Capacitor
npm install @capacitor/core @capacitor/cli

# 初始化配置
npx cap init

# 添加平台
npx cap add android
npx cap add ios
```

#### 第二阶段：原生适配 (5 天)

- 配置原生插件
- 适配原生功能
- 测试原生 API

#### 第三阶段：鸿蒙适配 (5 天)

- Web view 封装
- 鸿蒙特性适配
- 鸿蒙测试

#### 第四阶段：应用打包 (3 天)

- Android 打包
- iOS 打包
- 鸿蒙打包

#### 第五阶段：商店上架 (5 天)

- Google Play 上架
- App Store 上架
- 鸿蒙应用市场上架
