## 客户管理

### 新建客户

1. 点击"客户管理"菜单
2. 点击"新建客户"按钮
3. 填写客户信息：
   - 客户名称 (必填)
   - 客户类型 (企业/个人/政府)
   - 行业
   - 联系人
   - 联系电话
   - 邮箱
   - 省份/城市
   - 客户等级 (A/B/C)
4. 点击"提交"

### 编辑客户

1. 在客户列表中找到目标客户
2. 点击"编辑"按钮
3. 修改信息
4. 点击"保存"

### 删除客户

1. 在客户列表中找到目标客户
2. 点击"删除"按钮
3. 确认删除

### 搜索客户

在搜索框输入：
- 客户名称
- 联系人
- 联系电话

支持模糊搜索。

### 客户等级说明

| 等级 | 说明 | 跟进频率 |
|-----|------|---------|
| A 级 | 高意向客户 | 每周跟进 |
| B 级 | 中意向客户 | 每两周跟进 |
| C 级 | 低意向客户 | 每月跟进 |

## 线索管理

### 线索来源

- 官网询价
- 展会收集
- 客户推荐
- 广告投放
- 手动创建

### 新建线索

1. 点击"线索管理"菜单
2. 点击"新建线索"按钮
3. 填写线索信息：
   - 姓名 (必填)
   - 手机号 (必填)
   - 邮箱
   - 公司
   - 意向产品
   - 预算
   - 来源
4. 点击"提交"

### 线索状态

| 状态 | 说明 |
|-----|------|
| 新线索 | 刚创建的线索 |
| 已联系 | 已与客户联系 |
| 已确认 | 确认有需求 |
| 已转化 | 转化为客户 |
| 已流失 | 确认无需求 |

### 线索转化

1. 在线索列表中找到目标线索
2. 点击"转化"按钮
3. 系统自动创建客户档案
4. 线索状态变为"已转化"

### 线索分配

1. 管理员点击线索
2. 选择"分配"
3. 选择销售人员
4. 确认分配

## 商机管理

### 新建商机

1. 点击"商机管理"菜单
2. 点击"新建商机"按钮
3. 填写商机信息：
   - 商机名称
   - 关联客户
   - 预计金额
   - 成交概率
   - 预计成交日期
4. 点击"提交"

### 商机阶段

| 阶段 | 说明 | 概率 |
|-----|------|------|
| 初步接触 | 刚接触客户 | 10% |
| 需求分析 | 了解客户需求 | 30% |
| 方案报价 | 提供方案和报价 | 50% |
| 商务谈判 | 价格谈判 | 70% |
| 已成交 | 签约完成 | 100% |
| 已流失 | 客户放弃 | 0% |

### 更新商机阶段

1. 在商机列表中找到目标商机
2. 点击"编辑"
3. 修改阶段
4. 保存

### 销售漏斗

在仪表盘查看销售漏斗：
- 各阶段商机数量
- 各阶段预计金额
- 转化 rate 分析

## 订单管理

### 新建订单

1. 点击"订单管理"菜单
2. 点击"新建订单"按钮
3. 填写订单信息：
   - 客户
   - 产品明细
   - 数量
   - 单价
   - 总金额
   - 交付地址
   - 预计交付日期
4. 点击"提交"

### 订单状态

| 状态 | 说明 |
|-----|------|
| 待处理 | 刚创建的订单 |
| 已确认 | 客户确认订单 |
| 生产中 | 正在生产 |
| 待发货 | 生产完成 |
| 已发货 | 已发货 |
| 已完成 | 客户签收 |

### 订单统计

在仪表盘查看：
- 总订单数
- 待处理订单
- 已完成订单
- 销售收入

### 订单导出

1. 在订单列表页面
2. 点击"导出"按钮
3. 选择导出格式 (Excel/CSV)
4. 下载文件

## 系统设置

### 基本设置

1. 点击"系统设置"菜单
2. 填写公司信息：
   - 网站名称
   - 网站标题
   - 联系电话
   - 联系邮箱
   - 公司地址
3. 点击"保存"

### 用户管理

#### 新建用户

1. 点击"用户管理"菜单
2. 点击"新建用户"按钮
3. 填写用户信息：
   - 用户名
   - 邮箱
   - 密码
   - 手机号
   - 角色
   - 状态
4. 点击"提交"

#### 角色说明

| 角色 | 权限 |
|-----|------|
| 管理员 | 全部权限 |
| 经理 | 查看/编辑所有数据 |
| 销售 | 查看/编辑自己的数据 |
| 客服 | 查看权限 |

### 角色管理

1. 点击"角色管理"菜单
2. 可以创建自定义角色
3. 配置角色权限：
   - 客户管理
   - 线索管理
   - 商机管理
   - 订单管理
   - 产品管理
   - 系统设置

### 第三方集成

#### 钉钉集成

1. 点击"第三方集成"菜单
2. 点击"添加集成"
3. 选择"钉钉"
4. 输入 Webhook 地址
5. 选择通知事件
6. 点击"测试"
7. 保存

#### 企业微信集成

1. 点击"第三方集成"菜单
2. 点击"添加集成"
3. 选择"企业微信"
4. 输入 Webhook 地址
5. 选择通知事件
6. 点击"测试"
7. 保存

## 常见问题

### 忘记密码怎么办？

联系管理员重置密码。

### 如何修改个人信息？

1. 点击右上角头像
2. 选择"个人资料"
3. 修改信息
4. 保存

### 如何导出数据？

在列表页面点击"导出"按钮，选择格式后下载。

### 客户重复了怎么办？

1. 在客户列表搜索重复客户
2. 删除或合并重复客户
3. 建议新建时先搜索确认

### 线索如何分配给销售？

1. 管理员打开线索详情
2. 点击"分配"
3. 选择销售人员
4. 确认

### 如何查看销售业绩？

在仪表盘查看个人/团队销售数据。

### 系统支持哪些浏览器？

- Chrome (推荐)
- Firefox
- Safari
- Edge

### 手机端可以使用吗？

支持手机浏览器访问，系统已做响应式适配。
