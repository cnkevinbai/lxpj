## 快速入门

### 1. 登录系统

访问系统地址，输入用户名和密码登录。

**默认账号**:
- 管理员：admin / admin123
- 销售员：sales / sales123

### 2. 系统首页

登录后进入仪表盘，显示：
- 📊 核心业务指标
- 📋 待办事项
- 📈 最新动态

### 3. 快捷操作

| 快捷键 | 功能 |
|-------|------|
| `Ctrl + K` | 全局搜索 |
| `Ctrl + N` | 快速创建 |
| `Ctrl + S` | 保存 |

## CRM 系统

### 线索管理

#### 创建线索

1. 点击"线索" → "新建线索"
2. 填写客户信息
3. 选择线索来源
4. 点击"保存"

#### 线索跟进

1. 打开线索详情
2. 点击"添加跟进"
3. 填写跟进内容
4. 设置下次联系时间

#### 线索转化

1. 线索状态改为"有效"
2. 点击"转化为客户"
3. 确认转化

### 客户管理

#### 客户列表

支持筛选条件：
- 客户等级 (A/B/C)
- 行业分类
- 地区
- 负责人

#### 客户详情

查看客户完整信息：
- 基本信息
- 联系记录
- 商机记录
- 订单记录
- 合同记录

#### 公海池

**领取客户**:
1. 进入"公海池"
2. 选择客户
3. 点击"领取"

**释放客户**:
1. 打开客户详情
2. 点击"释放到公海"
3. 填写释放原因

### 商机管理

#### 创建商机

1. 点击"商机" → "新建商机"
2. 选择客户
3. 填写商机信息
4. 设置预计金额
5. 选择商机阶段

#### 商机阶段

| 阶段 | 说明 | 赢单率 |
|-----|------|--------|
| 初步接触 | 首次沟通 | 10% |
| 需求分析 | 明确需求 | 30% |
| 方案报价 | 提交方案 | 50% |
| 谈判协商 | 价格谈判 | 70% |
| 赢单 | 签订合同 | 100% |
| 输单 | 未成交 | 0% |

### 订单管理

#### 创建订单

1. 点击"订单" → "新建订单"
2. 选择客户和商机
3. 添加产品明细
4. 确认订单金额
5. 提交审批

#### 订单跟踪

订单状态流转：
```
草稿 → 待审批 → 已批准 → 生产中 → 已发货 → 已完成
```

## ERP 系统

### 采购管理

#### 创建采购单

1. 点击"采购" → "新建采购"
2. 选择供应商
3. 添加产品明细
4. 确认采购金额
5. 提交审批

#### 采购审批

1. 进入"待审批"
2. 查看采购详情
3. 点击"批准"或"拒绝"
4. 填写审批意见

### 库存管理

#### 入库管理

1. 点击"库存" → "入库"
2. 选择采购单
3. 确认入库数量
4. 选择库位
5. 确认入库

#### 出库管理

1. 点击"库存" → "出库"
2. 选择出库原因
3. 选择产品
4. 填写出库数量
5. 确认出库

#### 库存调拨

1. 点击"库存" → "调拨"
2. 选择源仓库
3. 选择目标仓库
4. 添加调拨产品
5. 提交审批

### 生产管理

#### 生产计划

1. 点击"生产" → "生产计划"
2. 点击"新建计划"
3. 选择产品
4. 填写计划数量
5. 设置计划时间

#### 生产任务

1. 点击"生产" → "生产任务"
2. 选择生产计划
3. 分配工作中心
4. 分配负责人
5. 开始生产

### 财务管理

#### 收款管理

1. 点击"财务" → "收款"
2. 点击"登记收款"
3. 选择订单
4. 填写收款金额
5. 选择收款方式

#### 付款管理

1. 点击"财务" → "付款"
2. 点击"登记付款"
3. 选择采购单
4. 填写付款金额
5. 选择付款方式

## 合同管理

### 创建合同

#### 方式一：从模板创建

1. 点击"合同" → "新建合同"
2. 选择合同模板
3. 填写合同信息
4. 系统自动生成合同内容

#### 方式二：自定义创建

1. 点击"合同" → "新建合同"
2. 选择"自定义"
3. 填写合同条款
4. 上传合同文件

### 合同审批

1. 提交合同审批
2. 选择审批平台 (内部/钉钉)
3. 审批人审批
4. 查看审批进度

### 电子签名

1. 合同审批通过
2. 点击"发起签署"
3. 上传签名/印章
4. 完成签署

### 合同归档

1. 合同生效
2. 点击"归档"
3. 选择归档分类
4. 完成归档

### 合同提醒

系统自动提醒：
- 到期前 30 天
- 到期前 15 天
- 到期前 7 天
- 到期前 3 天
- 到期前 1 天

## 审批流程

### 创建审批

#### 使用模板

1. 点击"审批" → "新建审批"
2. 选择审批模板
3. 填写审批信息
4. 提交审批

#### 自定义流程

1. 点击"审批" → "新建审批"
2. 选择"自定义"
3. 配置审批节点
4. 设置审批人
5. 提交审批

### 审批操作

#### 审批通过

1. 进入"待我审批"
2. 打开审批详情
3. 点击"通过"
4. 填写审批意见

#### 审批拒绝

1. 进入"待我审批"
2. 打开审批详情
3. 点击"拒绝"
4. 填写拒绝原因

### 审批平台

- 系统内审批
- 实时通知
- 自动流转

#### 钉钉审批

- 钉钉 APP 审批
- 消息推送
- 状态同步

#### 企业微信

- 企业微信审批
- 消息通知

## 常见问题

### 账号问题

**Q: 忘记密码怎么办？**

A: 点击登录页"忘记密码"，通过邮箱重置密码。

**Q: 如何修改个人信息？**

A: 点击右上角头像 → "个人设置" → 修改信息。

### 业务问题

**Q: 客户如何转为公海？**

A: 打开客户详情 → 点击"释放到公海" → 填写原因。

**Q: 审批流程如何配置？**

A: 使用预设模板或自定义配置，支持可视化编辑。

**Q: 合同到期如何提醒？**

A: 系统自动提醒，也可手动设置提醒时间。

### 技术问题

**Q: 页面加载慢怎么办？**

A: 清除浏览器缓存，或联系管理员检查服务器。

**Q: 数据导出失败？**

A: 检查数据量是否过大，建议分批次导出。
