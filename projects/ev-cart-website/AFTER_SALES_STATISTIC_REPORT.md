## 📊 报表总览

## 📈 统计维度

### 1. 总体统计

| 指标 | 说明 | 计算方式 |
|-----|------|---------|
| **总工单数** | 统计期内工单总数 | COUNT(*) |
| **完成率** | 已完成工单占比 | 已完成/总数 × 100% |
| **平均响应时间** | 从创建到分配的平均时间 | AVG(assignedAt - createdAt) |
| **平均满意度** | 客户评分平均值 | AVG(satisfaction) |

### 2. 工单类型分布

| 类型 | 说明 | 统计内容 |
|-----|------|---------|
| **安装工单** | installation | 数量/占比 |
| **维修工单** | repair | 数量/占比 |
| **保养工单** | maintenance | 数量/占比 |
| **退换货工单** | return | 数量/占比 |
| **咨询工单** | consultation | 数量/占比 |
| **投诉工单** | complaint | 数量/占比 |

### 3. 工单状态分布

| 状态 | 说明 | 统计内容 |
|-----|------|---------|
| **待受理** | pending | 当前数量 |
| **已受理** | accepted | 当前数量 |
| **已分配** | assigned | 当前数量 |
| **处理中** | processing | 当前数量 |
| **已完成** | completed | 累计数量 |
| **已关闭** | closed | 累计数量 |

### 4. 服务人员绩效

| 指标 | 说明 | 计算方式 |
|-----|------|---------|
| **总工单数** | 接单总数 | COUNT(*) |
| **完成工单数** | 已完成数 | COUNT(status='completed') |
| **完成率** | 完成占比 | 完成数/总数 × 100% |
| **平均满意度** | 客户评分 | AVG(satisfaction) |
| **平均响应时间** | 接单响应速度 | AVG(响应时间) |
| **绩效评级** | 综合评级 | 优秀/良好/一般/差 |

### 5. 满意度分布

| 评分 | 说明 | 统计内容 |
|-----|------|---------|
| **5 星** | 非常满意 | 数量/占比 |
| **4 星** | 满意 | 数量/占比 |
| **3 星** | 一般 | 数量/占比 |
| **2 星** | 不满意 | 数量/占比 |
| **1 星** | 非常不满意 | 数量/占比 |

### 6. 时间趋势

| 维度 | 说明 | 统计内容 |
|-----|------|---------|
| **日趋势** | 每日工单变化 | 总工单/已完成 |
| **周趋势** | 每周工单变化 | 总工单/已完成 |
| **月趋势** | 每月工单变化 | 总工单/已完成 |

## 🎯 API 接口

### 统计接口

| 方法 | 路径 | 说明 | 参数 |
|-----|------|------|------|
| GET | `/after-sales/stats/overview` | 总体统计 | startDate, endDate |
| GET | `/after-sales/stats/technician` | 人员绩效 | startDate, endDate |
| GET | `/after-sales/stats/daily` | 每日趋势 | days |
| GET | `/after-sales/stats/type-distribution` | 类型分布 | startDate, endDate |
| GET | `/after-sales/stats/satisfaction` | 满意度分布 | startDate, endDate |
| GET | `/after-sales/stats/response-time` | 响应时间 | startDate, endDate |
| GET | `/after-sales/stats/report` | 完整报表 | startDate, endDate |

## 🎯 绩效评级标准

### 评级规则

| 评级 | 满意度 | 完成率 | 响应时间 |
|-----|--------|--------|---------|
| **优秀** | ≥ 4.8 | ≥ 90% | ≤ 2 小时 |
| **良好** | ≥ 4.5 | ≥ 85% | ≤ 4 小时 |
| **一般** | ≥ 4.0 | ≥ 75% | ≤ 8 小时 |
| **差** | < 4.0 | < 75% | > 8 小时 |

### 综合评分计算

```typescript
综合评分 = (
  满意度得分 × 40% +
  完成率得分 × 40% +
  响应时间得分 × 20%
)

满意度得分 = 满意度 × 20 (满分 100)
完成率得分 = 完成率
响应时间得分 = MAX(0, 100 - 响应时间分钟数)
```
