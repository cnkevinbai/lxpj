## 📊 页面功能详情

### 工单列表页

**功能**:
- ✅ 工单列表展示
- ✅ 多条件筛选 (类型/状态/优先级)
- ✅ 工单号搜索
- ✅ 状态标签 (颜色区分)
- ✅ 快速查看
- ✅ 新建工单入口

**筛选条件**:
- 工单类型 (安装/维修/保养/退换货/咨询/投诉)
- 工单状态 (待受理/已受理/已分配/处理中/已完成/已关闭)
- 优先级 (普通/紧急/特急)

### 工单详情页

**功能**:
- ✅ 工单完整信息展示
- ✅ 服务进度时间线
- ✅ 联系客户按钮
- ✅ 完成工单操作
- ✅ 状态流转展示

**信息展示**:
- 基础信息 (工单号/类型/状态/优先级)
- 客户信息 (姓名/电话/地址)
- 产品信息 (名称/序列号)
- 问题描述
- 服务人员
- 时间戳

### 工单分配页 (主管专用)

**功能**:
- ✅ 待分配工单列表
- ✅ 多选工单 (批量分配)
- ✅ 技术人员选择 (显示技能/评分)
- ✅ 分配确认弹窗
- ✅ 自动通知提示

**分配流程**:
1. 选择待分配工单
2. 选择技术人员
3. 确认分配
4. 系统自动通知

## 📱 页面路由设计

```
/after-sales/
├── tickets/              # 工单管理
│   ├── list             # 工单列表
│   ├── new              # 新建工单
│   ├── assign           # 工单分配 (主管)
│   └── :id              # 工单详情
├── centers/             # 服务网点
│   ├── list             # 网点列表
│   ├── new              # 新建网点
│   ├── map              # 网点地图
│   └── :id              # 网点详情
├── contracts/           # 服务合同
│   ├── list             # 合同列表
│   ├── new              # 新建合同
│   ├── expiring         # 到期提醒
│   └── :id              # 合同详情
├── parts/               # 备件管理
│   ├── list             # 备件列表
│   ├── new              # 新建备件
│   ├── low-stock        # 库存预警
│   └── :id              # 备件详情
├── complaints/          # 投诉管理
│   ├── list             # 投诉列表
│   └── :id              # 投诉详情
└── statistics/          # 统计报表
    └── dashboard        # 统计看板
```
