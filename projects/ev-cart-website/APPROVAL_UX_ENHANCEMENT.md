## 📋 使用示例

### 1. 使用模板创建流程

```typescript
// 获取推荐模板
const templates = await http.get('/approval-flows/templates?recommended=true')

// 创建流程
const flow = await http.post('/approval-flows/templates/contract-simple/create', {
  id: 'user123',
  name: '张三',
})
```

### 2. 可视化流程

```typescript
// 获取流程图数据
const visual = await http.get('/approval-flows/:id/visualize')

// 返回
{
  flow: {...},
  visualData: {
    nodes: [...],
    edges: [...],
    width: 1000,
    height: 600
  },
  asciiPreview: "┌────────┐...",
  estimatedTime: 72,  // 小时
  complexity: 5  // 1-10 分
}
```

### 3. 验证配置

```typescript
// 验证流程配置
const result = await http.post('/approval-flows/validate', {
  nodes: [
    { order: 1, type: 'approver', name: '经理审批' },
    { order: 2, type: 'approver', name: '财务审批' }
  ]
})

// 返回
{
  valid: true,
  errors: [],
  warnings: ['节点 2：未指定审批人']
}
```
