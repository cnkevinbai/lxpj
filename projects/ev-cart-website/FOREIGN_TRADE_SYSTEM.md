## 📊 系统概览

道达智能外贸系统覆盖国际贸易全流程，从客户开发到订单执行、从报关到退税，实现外贸业务一体化管理。

### 核心模块
| 模块 | 功能 | 状态 |
|------|------|------|
| 外贸客户 | 国外客户档案、分级管理 | ✅ |
| 外贸询盘 | 询盘登记、跟进、转化 | ✅ |
| 外贸报价 | 多币种报价、价格计算 | ✅ |
| 外贸订单 | 出口订单管理 | ✅ |
| 出口报关 | 报关单据、报关进度 | ✅ |
| 装运管理 | 订舱、装运跟踪 | ✅ |
| 单证管理 | 发票、箱单、提单 | ✅ |
| 出口退税 | 退税申报、进度跟踪 | ✅ |

### 1. 外贸客户管理 (Foreign Customer Management)

**路径**: `/foreign-customers`  
**后端**: `backend/src/modules/foreign-customer/`

#### 核心功能

##### 1.1 客户档案
| 字段 | 说明 |
|------|------|
| 客户编号 | 唯一标识 |
| 客户名称 | 公司全称 (英文) |
| 国家/地区 | 所在国家/地区 |
| 联系地址 | 详细英文地址 |
| 联系人 | 主要联系人姓名 |
| 职位 | 联系人职位 |
| 电话 | 国际电话格式 |
| 邮箱 | 企业邮箱 |
| 网站 | 公司网站 |
| 客户来源 | 展会/阿里/独立站/推荐 |
| 客户等级 | A/B/C 分级 |
| 主营产品 | 感兴趣的产品类别 |
| 年采购量 | 预估年采购金额 |
| 付款方式 | T/T、L/C 等 |
| 信用额度 | 授信额度 (USD) |

##### 1.2 客户分级
| 等级 | 标准 | 政策 |
|------|------|------|
| A 类 | 年采购>$500K | 优先支持、账期 60 天 |
| B 类 | 年采购$100K-500K | 正常支持、账期 30 天 |
| C 类 | 年采购<$100K | 标准支持、款到发货 |

##### 1.3 客户跟进
| 功能 | 说明 | 状态 |
|------|------|------|
| 跟进记录 | 记录每次沟通内容 | ✅ |
| 跟进计划 | 设定下次跟进时间 | ✅ |
| 跟进提醒 | 到期自动提醒 | ✅ |
| 公海池 | 未跟进客户自动释放 | ✅ |

**路径**: `/foreign-inquiries`  
**后端**: `backend/src/modules/foreign-inquiry/`

#### 核心功能

##### 2.1 询盘登记
| 字段 | 说明 |
|------|------|
| 询盘编号 | 唯一标识 |
| 客户名称 | 询盘客户 |
| 询盘日期 | 收到询盘日期 |
| 询盘来源 | 阿里/官网/展会/邮件 |

| 目的港 | 运输目的港 |
| 付款方式 | 期望付款方式 |
| 询盘状态 | 新建/跟进中/已报价/已成交/已关闭 |

##### 2.2 询盘跟进
| 功能 | 说明 | 状态 |
|------|------|------|
| 分配业务员 | 询盘分配给业务员 | ✅ |
| 跟进记录 | 记录沟通进展 | ✅ |
| 报价记录 | 关联报价单 | ✅ |
| 转化订单 | 询盘转订单 | ✅ |
| 询盘分析 | 转化率统计 | ✅ |

##### 2.3 询盘转化漏斗
```
询盘总数 → 已联系 → 已报价 → 样品确认 → 成交订单

转化率:
- 联系率：已联系 / 询盘总数 (目标：≥ 90%)
- 报价率：已报价 / 已联系 (目标：≥ 60%)
- 成交率：成交订单 / 已报价 (目标：≥ 20%)
```

### 3. 外贸报价管理 (Quotation Management)

**路径**: `/foreign-quotations`  
**后端**: `backend/src/modules/foreign-inquiry/`

#### 核心功能

##### 3.1 报价计算
| 项目 | 说明 |
|------|------|
| 产品单价 | FOB/CIF/CFR 单价 |
| 数量 | 订单数量 |
| 金额 | 单价 × 数量 |
| 运费 | 海运费/空运费 |
| 保险费 | 货运保险 |
| 杂费 | 报关费、文件费等 |
| 总价 | 合计金额 |
| 币种 | USD/EUR/GBP 等 |
| 汇率 | 实时汇率 |
| 有效期 | 报价有效期 |

##### 3.2 价格术语
| 术语 | 含义 | 责任 |
|------|------|------|
| EXW | 工厂交货 | 买方承担所有运费和风险 |
| FOB | 船上交货 | 卖方负责到装运港 |
| CIF | 成本 + 保险 + 运费 | 卖方负责到目的港 |
| DDP | 完税后交货 | 卖方负责到买方指定地点 |

##### 3.3 报价单模板
```

│                      QUOTATION                              │
│  报价单号：QT-2026-001                                      │

│  有效期：30 天                                              │
├─────────────────────────────────────────────────────────────┤
│  To: [Customer Name]                                        │
│  From: [Company Name]                                       │
├─────────────────────────────────────────────────────────────┤
│  Product | Quantity | Unit Price | Amount                   │
│  --------|----------|------------|--------                  │
│  Item 1  | 1000 PCS | USD 10.00  | USD 10,000              │
│  Item 2  | 500 PCS  | USD 15.00  | USD 7,500               │
├─────────────────────────────────────────────────────────────┤
│  Total Amount: USD 17,500                                   │
│  Payment Terms: 30% T/T in advance, 70% before shipment     │
│  Delivery Time: 30 days after receiving deposit             │
│  Port of Loading: Shanghai/Ningbo                           │
└─────────────────────────────────────────────────────────────┘
```

```
- 纺织品：13%
- 塑料制品：13%

```

## 🔄 核心业务流程

### 外贸订单全流程
```
客户开发 → 询盘 → 报价 → 样品确认 → 订单签订 → 收取定金
    ↓
生产下达 → 生产跟踪 → 质检 → 订舱 → 报关 → 发货
    ↓
提交单据 → 收取尾款 → 出口退税 → 订单完成
```

### 收款流程
```
订单签订 → 收取定金 (30%) → 生产完成 → 发货前收尾款 (70%) → 退税
```

## 📈 关键指标 (KPI)

| 指标 | 公式 | 目标值 |
|------|------|--------|
| 询盘转化率 | 成交订单数 / 询盘总数 | ≥ 20% |
| 订单及时交付率 | 按时交付订单 / 总订单 | ≥ 95% |
| 收汇及时率 | 按时收汇金额 / 应收汇金额 | ≥ 90% |
| 退税完成率 | 已完成退税 / 应退税订单 | ≥ 95% |
| 客户复购率 | 复购客户数 / 总客户数 | ≥ 40% |
| 毛利率 | (收入 - 成本) / 收入 | ≥ 25% |
