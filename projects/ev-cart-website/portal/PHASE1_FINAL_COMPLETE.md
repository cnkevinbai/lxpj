## 🎯 架构决策

### ✅ 统一门户架构（业财一体化）

**核心原则**:
- ✅ **业财一体化** - 业务单据自动生成财务凭证
- ✅ **业服一体化** - 服务工单关联销售订单
- ✅ **端到端自动化** - 订单→生产→采购→入库→出库→收款
- ✅ **数据实时同步** - 库存/成本/应收实时更新
- ✅ **统一平台** - 单点登录、统一权限、统一数据

**系统统一**:
```
portal/ (统一前端)
├── 官网 (对外展示)
├── CRM (客户/商机/订单)
├── ERP (生产/采购/库存/质量/设备/成本/资产/出口)
├── 财务 (应收/应付/费用/资金/预算/资产)
├── 售后 (服务工单/配件/知识库)
├── HR (员工/招聘/考勤/绩效/培训/薪酬)
├── CMS (新闻/案例/解决方案/视频)
├── 消息 (站内信/邮件/短信)
├── 工作流 (审批流程)
└── 系统管理 (用户/角色/权限)
```

- React Router 6.22

- Axios 1.6

### 代码结构
```
portal/src/
├── layouts/          # 布局组件
│   ├── WebsiteLayout.tsx
│   └── PortalLayout.tsx
├── pages/
│   ├── website/      # 官网页面
│   └── portal/       # 内部系统
│       ├── crm/      # CRM 模块
│       ├── erp/      # ERP 模块
│       ├── finance/  # 财务模块
│       ├── aftersales/ # 售后模块
│       ├── hr/       # HR 模块
│       └── ...
├── shared/           # 共享组件/服务
├── store/            # 状态管理
└── App.tsx           # 统一路由
```

### 路由设计

## 📈 一体化流程

### 业财一体化
```
销售订单 (CRM)
    ↓ 自动触发
生产工单 (ERP)
    ↓ 自动触发
采购申请 (ERP)
    ↓ 自动触发
入库/出库
    ↓ 自动生成
应收/应付 (财务)
    ↓ 自动生成
财务凭证
```

### 业服一体化
```
服务工单 (售后)
    ↓ 关联
销售订单 (CRM)
    ↓ 自动扣减
配件库存 (ERP)
    ↓ 自动结算
服务费用 (财务)
```
