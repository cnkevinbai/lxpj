## 🔄 完整邮寄流程

### 标准流程
```
1. 主管决策 → 寄件服务
   ↓
2. 创建邮寄单
   ├─ 配件清单
   ├─ 收货地址
   └─ 旧件回收（可选）
   ↓
3. 配件拣货
   ├─ 仓库拣货
   └─ 扫码核对
   ↓
4. 配件质检
   ├─ 质量检查
   └─ 拍照留证
   ↓
5. 配件打包
   ├─ 包装材料
   ├─ 填充物
   └─ 称重测量
   ↓
6. 快递下单
   ├─ 选择快递公司
   ├─ 获取运单号
   └─ 打印快递单
   ↓
7. 确认发货
   ├─ 扫描发货
   ├─ 通知客户
   └─ 更新工单
   ↓
8. 物流跟踪
   ├─ 自动同步
   ├─ 异常预警
   └─ 签收提醒
   ↓
9. 客户收货
   ├─ 确认收货
   ├─ 评价反馈
   └─ 异常处理
   ↓
10. 旧件回收（可选）
    ├─ 创建回收单
    ├─ 客户寄回
    ├─ 仓库收货
    └─ 旧件检验
```

## 🎯 智能功能

### 1. 快递智能推荐
```typescript
推荐规则:
- 距离 < 200km → 顺丰当日达
- 距离 < 500km → 顺丰次日达
- 距离 < 1000km → 中通/圆通
- 距离 > 1000km → 顺丰/EMS
- 贵重物品 → 顺丰保价
- 大件物品 → 德邦物流
```

### 2. 运费自动计算
```typescript
运费 = 首重价格 + (总重量 - 1) × 续重价格 + 偏远地区附加费
```

### 3. 物流异常预警
```typescript
预警类型:
- 延迟预警（超过预计时间 24 小时）
- 丢失预警（物流信息 48 小时未更新）
- 损坏预警（客户反馈损坏）
- 退回预警（无法投递退回）
```

### 4. 自动通知
```typescript
通知节点:
- 发货通知（短信 + APP）
- 物流更新通知（APP）
- 签收提醒（短信）
- 旧件回收提醒（短信）
```
