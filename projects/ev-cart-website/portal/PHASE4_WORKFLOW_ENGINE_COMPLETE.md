## 🎯 支持的审批流程

### 采购审批
```typescript
采购订单 → 部门经理审批 → (金额>10 万) 总经理审批 → 财务审批 → 采购执行
```

### 费用报销审批
```typescript
费用申请 → 部门经理审批 → (金额>5000) 财务审批 → (金额>1 万) 总经理审批 → 出纳付款
```

### 合同审批
```typescript
合同起草 → 法务审核 → 财务审核 → (金额>10 万) 总经理审批 → 盖章
```

### 请假审批
```typescript
请假申请 → 部门经理审批 → (天数>3) 人事备案 → 完成
```

### 生产工单审批
```typescript
生产工单 → 生产主管审批 → 物料检查 → 工艺审核 → 生产执行
```

## 🔧 使用示例

### 启动审批流程
```typescript
import { workflowEngineService } from '@/shared/services/workflow-engine'

// 启动采购订单审批
const instance = await workflowEngineService.startInstance(
  'purchase-approval-definition-id',
  {
    orderId: 'PO20260314001',
    amount: 150000,
    supplier: 'XX 供应商',
  },
  'PO20260314001'
)
```

### 审批通过
```typescript
import { approvalFlowService } from '@/shared/services/approval-flow'

// 审批通过
await approvalFlowService.approve(
  'task-id',
  '同意采购，价格合理'
)
```

### 创建定时任务
```typescript
import { schedulerService, CronExpression } from '@/shared/services/scheduler'

// 创建每日数据同步任务
await schedulerService.createTask({
  name: '每日数据同步',
  description: '每天凌晨 2 点同步 CRM 和 ERP 数据',
  cronExpression: CronExpression.daily(2, 0),
  handler: 'dataSyncHandler',
  params: {
    source: 'crm',
    target: 'erp',
  },
  enabled: true,
})
```
