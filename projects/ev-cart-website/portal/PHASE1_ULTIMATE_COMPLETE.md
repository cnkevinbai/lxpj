## 📈 一体化流程设计

### 业财一体化
```
销售订单 → 生产工单 → 采购申请 → 入库/出库 → 应收/应付 → 财务凭证
```

### 业服一体化
```
服务工单 → 关联订单 → 扣减库存 → 结算费用 → 财务凭证
```

### 端到端自动化
```
客户下单 → 库存检查 → 生产计划 → 采购申请 → 生产入库 → 出库发货 → 收款核销
```
