## 🚀 部署方案

### 方案一：传统部署
```bash
# 1. 数据库初始化
psql -U evcart -d evcart -f database/migrations/*.sql

# 2. 后端部署
cd backend && npm install && npm run build && npm run start:prod

# 3. 前端部署
cd crm && npm install && npm run build
# 将 dist/ 部署到 Nginx

# 4. Nginx 配置
# 配置反向代理到后端 API
```

### 方案二：Docker 部署
```bash
# 一键启动
docker-compose up -d

# 服务包含
- PostgreSQL 数据库
- NestJS 后端 API
- React 前端应用
```
