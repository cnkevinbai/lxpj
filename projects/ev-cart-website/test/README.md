## 📋 测试命令

### 运行所有测试

```bash
# 根目录
npm run test

# 后端测试
cd backend && npm run test

# 前端测试
cd crm && npm run test

# E2E 测试
npm run test:e2e

# 性能测试
npm run test:perf
```

### 生成覆盖率报告

```bash
# 后端覆盖率
cd backend && npm run test:cov

# 前端覆盖率
cd crm && npm run test:cov
```

### 查看覆盖率报告

```bash
# 后端
open backend/coverage/lcov-report/index.html

# 前端
open crm/coverage/lcov-report/index.html
```
