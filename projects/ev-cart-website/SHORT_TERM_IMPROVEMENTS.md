### 导入类型

| 类型 | 必填字段 | 状态 |
|-----|---------|------|
| 线索导入 | 姓名、手机 | ✅ |
| 客户导入 | 客户名称 | ✅ |

### API 接口

| 接口 | 方法 | 说明 |
|-----|------|------|
| `/import/leads` | POST | 导入线索 |
| `/import/customers` | POST | 导入客户 |
| `/import/template` | GET | 下载模板 |

### 使用方法

```typescript
// 上传文件
const file = new File([...], 'data.xlsx')
const formData = new FormData()
formData.append('file', file)

// 导入线索
await apiClient.post('/import/leads', formData, {
  params: { businessType: 'domestic' }
})
```
