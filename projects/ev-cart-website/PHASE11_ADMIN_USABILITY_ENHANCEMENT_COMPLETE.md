## 📝 1. 操作日志

### 日志记录内容

**记录字段**:
```typescript
{
  userId: string,           // 用户 ID
  userName: string,         // 用户姓名
  userDepartment: string,   // 用户部门
  module: string,           // 模块
  action: string,           // 操作
  actionName: string,       // 操作名称
  status: 'success' | 'failed' | 'error',
  targetId: string,         // 目标对象 ID
  targetName: string,       // 目标对象名称
  requestUrl: string,       // 请求 URL
  requestMethod: string,    // 请求方法
  requestBody: any,         // 请求参数
  responseData: any,        // 响应数据
  ipAddress: string,        // IP 地址
  userAgent: string,        // 浏览器信息
  browser: string,          // 浏览器
  os: string,               // 操作系统
  device: string,           // 设备类型
  duration: number,         // 耗时（毫秒）
  errorMessage: string,     // 错误信息
  createdAt: Date           // 操作时间
}
```

---

### 日志查询

**查询条件**:
- 🔍 关键词搜索（用户名/操作名/目标名）
- 📅 日期范围
- 👤 用户筛选
- 📊 模块筛选
- ✅ 状态筛选

**API**:
```typescript
// 获取操作日志
GET /api/v1/system/logs?page=1&limit=20&userId=xxx&module=user&status=success

// 获取用户统计
GET /api/v1/system/logs/statistics?userId=xxx&days=30

// 获取热门操作
GET /api/v1/system/logs/popular?limit=10
```

## 📥 3. 数据导入

### 导入功能

**支持导入**:
- 👥 用户批量导入
- 🏢 部门批量导入

**导入流程**:
```
1. 下载导入模板
   ↓
2. 填写数据
   ↓
3. 上传文件
   ↓
4. 数据验证
   ↓
5. 导入结果
```

### 导入 API

```typescript
// 获取导入模板
GET /api/v1/system/import/template?type=users

// 导入用户
POST /api/v1/system/import/users
Content-Type: multipart/form-data
File: users.xlsx

// 导入部门
POST /api/v1/system/import/departments
```
