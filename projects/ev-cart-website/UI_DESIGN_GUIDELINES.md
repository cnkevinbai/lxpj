## 设计原则

### 1. 一致性 (Consistency)

- ✅ 统一的视觉语言
- ✅ 一致的交互模式
- ✅ 标准化的组件库

### 2. 可用性 (Usability)

- ✅ 清晰的视觉层次
- ✅ 明确的交互反馈
- ✅ 易于理解的操作

### 3. 可访问性 (Accessibility)

- ✅ 支持键盘操作
- ✅ 色盲友好配色
- ✅ 屏幕阅读器支持

### 4. 响应式 (Responsive)

- ✅ 移动端优先
- ✅ 多端适配
- ✅ 流畅的动画

## 字体系统

### 中文字体

```css
font-family:
  'PingFang SC',           /* 苹果中文字体 */
  'Microsoft YaHei',       /* 微软雅黑 */
  'Noto Sans SC',          /* 思源黑体 */
  sans-serif;
```

### 英文字体

```css
font-family:
  'Inter',                 /* 现代无衬线 */
  'SF Pro Display',        /* 苹果字体 */
  'Helvetica Neue',
  sans-serif;
```
