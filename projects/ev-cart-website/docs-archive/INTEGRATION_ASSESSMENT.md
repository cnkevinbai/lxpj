### 集成架构

```
┌─────────────────────────────────────────────────────────┐
│                    道达智能 CRM                          │
│                  审批流程引擎                            │
└────────────────────┬────────────────────────────────────┘
                     │ Webhook / API
                     ▼
┌─────────────────────────────────────────────────────────┐
│                    钉钉开放平台                          │
│            审批实例创建 / 状态同步                       │
└─────────────────────────────────────────────────────────┘
```

### 核心功能

#### 1. 审批模板映射
- CRM 审批类型 → 钉钉审批模板
- 字段自动映射
- 审批人同步

#### 2. 审批流程
- 创建钉钉审批实例
- 推送审批数据
- 同步审批状态
- 接收审批结果

#### 3. 通知推送
- 审批待办通知
- 审批结果通知
- 消息提醒
