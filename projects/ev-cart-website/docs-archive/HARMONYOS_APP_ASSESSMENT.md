## 🎯 鸿蒙应用设计方案

### 方案一：基于现有 API 重构 (推荐)

**架构**:
```
┌─────────────────────────────────────────────────────────┐
│              鸿蒙原生应用 (ArkTS)                        │
├──────────────┬──────────────┬──────────────────────────┤
│   CRM 模块   │   审批模块   │   消息模块               │
│  (ArkUI)     │  (ArkUI)     │  (ArkUI)                 │
└──────────────┴──────────────┴──────────────────────────┘
                     │
                     ▼
         现有后端 API (复用)
```

**优势**:
- ✅ 后端 API 完全复用
- ✅ 数据一致性保证
- ✅ 开发成本可控
- ✅ 上线周期短 (2-3 月)

**劣势**:
- ⚠️ 需要前端重构
- ⚠️ 维护两套前端代码

**预计工作量**:
- UI 设计：2 周
- 核心功能：6 周
- 测试优化：2 周
- **总计**: 10 周

### 方案二：跨平台方案 (Uni-app/Taro)

**架构**:
```
┌─────────────────────────────────────────────────────────┐
│              跨平台应用 (Vue/React)                      │
├──────────────┬──────────────┬──────────────────────────┤
│   iOS        │   Android    │   鸿蒙                    │
│  (App)       │  (App)       │  (App)                   │
└──────────────┴──────────────┴──────────────────────────┘
```

**优势**:
- ✅ 一套代码多端运行
- ✅ 开发成本低
- ✅ 维护简单

**劣势**:
- ⚠️ 性能略低于原生
- ⚠️ 鸿蒙特性支持有限

**预计工作量**:
- 适配开发：4 周
- 测试优化：2 周
- **总计**: 6 周

### 核心页面

#### 1. 首页

```typescript
@Entry
@Component
struct Index {
  @State quickActions: QuickAction[] = []
  @State pendingApprovals: number = 0
  @State todayTasks: Task[] = []

  build() {
    Column() {
      // 顶部搜索
      SearchBar()
      
      // 快捷操作
      Grid() {
        ForEach(this.quickActions, (action) => (
          QuickActionButton(action)
        ))
      }
      
      // 待审批
      PendingApprovals(this.pendingApprovals)
      
      // 今日任务
      TaskList(this.todayTasks)
    }
  }
}
```

#### 2. 客户列表

```typescript
@Entry
@Component
struct CustomerList {
  @State customers: Customer[] = []
  @State loading: boolean = false

  aboutToAppear() {
    this.loadCustomers()
  }

  async loadCustomers() {
    this.loading = true
    this.customers = await CustomerService.getList()
    this.loading = false
  }

  build() {
    Column() {
      if (this.loading) {
        LoadingIndicator()
      } else {
        List() {
          ForEach(this.customers, (customer) => (
            CustomerItem(customer)
              .onClick(() => {
                router.pushUrl({ url: `/customer/${customer.id}` })
              })
          ))
        }
      }
    }
  }
}
```

#### 3. 审批详情

```typescript
@Entry
@Component
struct ApprovalDetail {
  @Param approvalId: string
  @State approval: Approval
  @State comments: string = ''

  build() {
    Column() {
      // 审批信息
      ApprovalInfo(this.approval)
      
      // 审批流程
      ApprovalFlow(this.approval.flow)
      
      // 审批操作
      Row() {
        Button('拒绝')
          .type(ButtonType.Danger)
          .onClick(() => this.reject())
        
        Button('通过')
          .type(ButtonType.Primary)
          .onClick(() => this.approve())
      }
    }
  }
}
```
