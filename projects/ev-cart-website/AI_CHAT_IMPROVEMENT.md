## 📋 实体结构

### ChatMessage

| 字段 | 类型 | 说明 |
|-----|------|------|
| id | UUID | 消息 ID |
| userId | string | 用户 ID |
| sender | string | 发送者 (user/bot) |
| content | text | 消息内容 |
| rating | number | 满意度评分 |
| createdAt | Date | 创建时间 |

## 🔌 API 接口

| 接口 | 方法 | 权限 | 说明 |
