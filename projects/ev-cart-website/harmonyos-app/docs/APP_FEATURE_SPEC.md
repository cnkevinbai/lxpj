## UI 风格

### 设计风格

**设计理念**: 简洁、高效、专业

- **简洁**: 界面清爽，信息层次清晰
- **高效**: 操作便捷，三步内完成核心操作
- **专业**: 符合企业级应用规范

### 视觉元素

#### 圆角规范

| 元素 | 圆角 | 示例 |
|-----|------|------|
| 按钮 | 24dp | 胶囊形 |
| 卡片 | 12dp | 圆角矩形 |
| 输入框 | 8dp | 小圆角 |
| 图标 | 4dp | 微圆角 |

#### 阴影规范

| 层级 | 阴影值 | 用途 |
|-----|--------|------|
| 基础 | `0 2px 8px rgba(0,0,0,0.05)` | 卡片 |
| 悬浮 | `0 4px 16px rgba(0,0,0,0.1)` | 悬浮按钮 |
| 弹窗 | `0 8px 32px rgba(0,0,0,0.15)` | 对话框 |

#### 动效规范

| 动效 | 曲线 | 时长 |
|-----|------|------|
| 页面切换 | EaseInOut | 300ms |
| 按钮点击 | EaseOut | 150ms |
| 列表加载 | Linear | 200ms |
| 弹窗出现 | EaseOut | 250ms |

### 暗黑模式

#### 色彩适配

## 性能优化

### 列表优化

```typescript
// 使用 LazyForEach 延迟渲染
LazyForEach(this.dataSource, (item) => {
  ListItem() {
    CustomerCard(item)
  }
}, (item) => item.id)
```

### 图片优化

```typescript
// 图片缓存
Image(url)
  .cachedImage({
    cache: CacheType.MEMORY
  })

// 图片懒加载
Image(url)
  .lazyLoad(true)
```

### 数据缓存

```typescript
// 本地缓存
Storage.get('customer_list', [])

// 内存缓存
AppStorage.getOrCreate('cache_key', data)
```
