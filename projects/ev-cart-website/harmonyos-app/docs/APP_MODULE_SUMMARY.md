## 🔐 权限体系

### 角色定义

| 角色 | 说明 | 权限范围 |
|-----|------|---------|
| **admin** | 系统管理员 | 全部权限 |
| **service_manager** | 售后主管 | 售后全部 + 工单分配 |
| **technician** | 服务人员 | 我的工单 + 处理 |
| **customer_service** | 客服人员 | 工单创建/受理 |
| **sales** | 销售人员 | CRM 全部 |
| **finance** | 财务人员 | 财务查看 |

### 权限控制

```typescript
// 权限检查
const checkPermission = (user: User, permission: string): boolean => {
  return user.permissions.includes(permission)
}

// 页面级权限
const routes = [
  {
    path: '/after-sales/tickets/assign',
    component: TicketAssign,
    permission: 'after_sales:ticket:assign',
  },
]

// 按钮级权限
<Button
  visible={checkPermission(user, 'after_sales:ticket:assign')}
>
  分配工单
</Button>
```

## 📊 数据流转

### CRM → 售后

```
销售成交 (CRM)
    ↓
客户信息 → 售后服务
    ↓
创建服务档案
    ↓
服务申请 → 工单创建
```

### 售后 → CRM

```
服务完成 (售后)
    ↓
服务记录 → 回流 CRM
    ↓
完善客户画像
    ↓
提升客户满意度
```
