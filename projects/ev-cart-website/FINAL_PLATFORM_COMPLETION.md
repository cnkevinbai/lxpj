## 📋 部署说明

### 后端部署
```bash
cd backend
npm install
cp .env.example .env
npm run prisma:generate
npm run prisma:migrate
npm run start:prod
```

### 前端部署
```bash
cd portal
npm install
npm run build
# 部署 dist 到 Nginx
```

### Docker 部署
```bash
docker-compose up -d
```

**访问地址**:
- 前端：http://localhost:5173
- 后端 API: http://localhost:3001/api
- Swagger: http://localhost:3001/api/docs
