## 📊 官网与业务系统联动方案

### 高优先级联动 (立即实现)

#### 1. 服务工单 ↔ 售后服务系统 🔴

**联动点**:
- 官网服务工单提交 → 创建售后工单
- 工单进度查询 → 读取售后系统数据

**API 接口**:
```javascript
POST /api/v1/service/tickets     // 创建服务工单
GET  /api/v1/service/tickets/:id  // 查询工单进度
```

**实现方案**:
```javascript
// 官网表单提交
const submitTicket = async (data) => {
  const response = await fetch('/api/v1/service/tickets', {
    method: 'POST',
    body: JSON.stringify(data),
  })
  // 自动创建售后工单，返回工单号
}
```

#### 3. 经销商加盟 ↔ 渠道管理系统 🔴

**联动点**:
- 加盟申请表单 → 创建经销商线索
- 加盟政策 → 读取渠道管理数据
- 区域查询 → 查询区域保护状态
- 申请进度 → 查询审核状态

**API 接口**:
```javascript
POST /api/v1/dealers/apply       // 提交加盟申请
GET  /api/v1/dealers/policy      // 获取加盟政策
GET  /api/v1/dealers/regions     // 查询区域状态
GET  /api/v1/dealers/application/:id  // 查询申请进度
```

**实现方案**:
```javascript
// 加盟申请
const submitDealerApply = async (data) => {
  const response = await fetch('/api/v1/dealers/apply', {
    method: 'POST',
    body: JSON.stringify(data),
  })
  // 返回申请 ID，用于后续查询进度
  return response.data.applicationId
}
```
