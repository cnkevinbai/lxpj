### 1. 线索管理 (Lead Management)

**路径**: `/leads`  
**后端**: `backend/src/modules/lead/`

#### 核心功能
| 功能 | 说明 | 状态 |
|------|------|------|
| 线索获取 | 官网/活动/导入/手动创建 | ✅ |
| 线索分配 | 自动分配/手动分配 | ✅ |
| 线索培育 | 跟进、邮件营销、内容推送 | ✅ |
| 线索转化 | 转化为商机/客户 | ✅ |
| 线索池 | 公海池、私海池 | ✅ |
| 线索统计 | 转化率、来源分析 | ✅ |

#### 线索来源
| 来源 | 说明 | 占比 |
|------|------|------|
| 官网咨询 | 官网表单、在线客服 | 30% |
| 市场活动 | 展会、研讨会、网络研讨会 | 25% |
| 阿里国际站 | 阿里巴巴平台询盘 | 20% |
| 客户推荐 | 老客户推荐 | 15% |
| 主动开发 | 销售自主开发 | 10% |

#### 线索转化漏斗
```
线索总数 → 已联系 → 有意向 → 商机 → 成交

转化率:
- 联系率：已联系 / 线索总数 (目标：≥ 90%)
- 意向率：有意向 / 已联系 (目标：≥ 40%)
- 转化率：商机 / 有意向 (目标：≥ 50%)
- 成交率：成交 / 商机 (目标：≥ 30%)
```

## 🔄 核心业务流程

### 线索到现金 (L2C)
```
线索获取 → 线索分配 → 线索培育 → 线索转化
    ↓
商机创建 → 商机推进 → 报价/谈判 → 赢单
    ↓
订单创建 → 订单审核 → 生产/发货 → 回款
```

### 客户生命周期
```
潜在客户 → 线索 → 商机 → 客户 → 成交客户 → 忠诚客户
```

## 📈 关键指标 (KPI)

| 指标 | 公式 | 目标值 |
|------|------|--------|
| 线索转化率 | 成交客户数 / 线索总数 | ≥ 5% |
| 商机赢率 | 赢单数 / 商机总数 | ≥ 30% |
| 销售目标达成率 | 实际销售额 / 目标销售额 | ≥ 90% |
| 客户满意度 | 满意客户数 / 总客户数 | ≥ 95% |
| 回款及时率 | 按时回款金额 / 应收金额 | ≥ 90% |
| 客户流失率 | 流失客户数 / 总客户数 | ≤ 5% |

## 🔧 技术实现

### 数据表结构
```sql
-- 线索表
CREATE TABLE crm_leads (
  id UUID PRIMARY KEY,
  source VARCHAR(50),
  name VARCHAR(100),
  company VARCHAR(200),
  phone VARCHAR(50),
  email VARCHAR(100),
  status VARCHAR(20),
  owner_id UUID,
  created_at TIMESTAMP,
  converted_at TIMESTAMP
);

-- 商机表
CREATE TABLE crm_opportunities (
  id UUID PRIMARY KEY,
  name VARCHAR(200),
  customer_id UUID,
  amount DECIMAL(18,2),
  stage VARCHAR(50),
  probability INT,
  close_date DATE,
  owner_id UUID,
  status VARCHAR(20),
  created_at TIMESTAMP
);

-- 客户表
CREATE TABLE crm_customers (
  id UUID PRIMARY KEY,
  name VARCHAR(200),
  type VARCHAR(20),
  industry VARCHAR(100),
  level VARCHAR(10),
  credit_limit DECIMAL(18,2),
  owner_id UUID,
  created_at TIMESTAMP
);

-- 订单表
CREATE TABLE crm_orders (
  id UUID PRIMARY KEY,
  order_number VARCHAR(50),
  customer_id UUID,
  amount DECIMAL(18,2),
  status VARCHAR(20),
  payment_status VARCHAR(20),
  delivery_status VARCHAR(20),
  created_at TIMESTAMP
);
```
