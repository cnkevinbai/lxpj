# 电动观光车官网 + CRM 系统

> 项目名称：EV Cart Website & CRM  
> 创建时间：2026-03-11  
> 开发者：渔晓白 ⚙️  
> 状态：🟡 开发中

---

## 📦 项目概述

**目标**: 为电动观光车公司开发一体化官网 + CRM 系统

| 模块 | 技术栈 | 说明 |
|-----|-------|------|
| **官网** | Next.js 14 + Tailwind CSS | 对外展示、产品宣传、询价表单 |
| **CRM** | React 18 + Ant Design Pro | 客户管理、线索跟进、订单管理 |
| **后端** | NestJS + PostgreSQL | 统一 API 服务、数据管理 |
| **部署** | Docker + 阿里云 | 容器化部署、国内访问优化 |

---

## 🎯 核心功能

### 官网部分

- [ ] 首页 (轮播、产品展示、核心优势)
- [ ] 产品中心 (车型列表、参数详情、对比功能)
- [ ] 解决方案 (景区、酒店、地产、工厂等场景)
- [ ] 案例展示 (客户案例、实景图片)
- [ ] 服务支持 (售后政策、配件查询)
- [ ] 关于我们 (公司介绍、团队、资质、发展历程)
- [ ] 联系我们 (询价表单、地图、联系方式)
- [ ] **经销商加盟** (加盟政策、区域保护、支持体系、在线申请) ⭐ 新增
- [ ] **人才招聘** (招聘岗位、企业文化、员工福利、在线投递) ⭐ 新增

### CRM 部分

- [ ] 用户权限 (角色管理、登录认证)
- [ ] 客户管理 (客户档案、联系人、跟进记录)
- [ ] 线索管理 (官网询价自动入库、线索分配)
- [ ] 商机管理 (销售阶段、报价、合同)
- [ ] 订单管理 (订单流程、生产进度、发货)
- [ ] 产品管理 (车型库、配置选项、价格表)
- [ ] 数据报表 (销售漏斗、业绩统计)
