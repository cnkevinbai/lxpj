- ✅ 频道订阅/取消订阅
- ✅ 工单通知推送
- ✅ 审批通知推送
- ✅ 消息通知推送

**使用示例**:
```typescript
// 客户端连接
const socket = io('ws://api.example.com', {
  query: { token: 'xxx' }
})

// 订阅频道
socket.emit('subscribe', {
  channels: ['tickets', 'approvals', 'messages']
})

// 接收通知
socket.on('notification', (data) => {
  console.log('收到通知:', data)
})

// 接收工单通知
socket.on('new_ticket', (ticket) => {
  console.log('新工单:', ticket)
})
```
