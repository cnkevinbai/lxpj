## 🔌 前端调用示例

### 服务工单提交
```javascript
// 官网服务支持页面
const submitTicket = async (formData) => {
  const response = await fetch('/api/v1/service/tickets', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(formData),
  })
  const result = await response.json()
  if (result.success) {
    message.success(`工单创建成功！工单号：${result.data.id}`)
  }
}
```

### 加盟申请提交
```javascript
// 官网经销商加盟页
const submitApplication = async (formData) => {
  const response = await fetch('/api/v1/dealers/apply', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(formData),
  })
  const result = await response.json()
  if (result.success) {
    message.success(`申请提交成功！申请号：${result.data.id}`)
    // 保存申请号用于后续查询进度
    localStorage.setItem('applicationId', result.data.id)
  }
}
```

### 产品列表加载
```javascript
// 官网产品中心页
const loadProducts = async () => {
  const response = await fetch('/api/v1/products')
  const result = await response.json()
  if (result.success) {
    setProducts(result.data)
  }
}
```
