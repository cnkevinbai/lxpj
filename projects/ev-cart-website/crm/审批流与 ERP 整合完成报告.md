```

1. 申请人提交 → 部门经理审批
2. 部门经理审批 → 财务经理审批
3. 财务经理审批 → 完成

合同审批流程 (4 步):
1. 申请人提交 → 部门经理审批
2. 部门经理审批 → 财务经理审批
3. 财务经理审批 → 法务审批
4. 法务审批 → 总经理审批 → 完成

费用报销流程 (2 步):
1. 申请人提交 → 部门经理审批
2. 部门经理审批 → 财务审批 → 完成
```
