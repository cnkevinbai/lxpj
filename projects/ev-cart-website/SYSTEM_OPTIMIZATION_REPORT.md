## 🚀 使用指南

### 全局搜索
```
1. 按 Ctrl+K 唤起搜索框
2. 输入关键词
3. 点击结果直接跳转
```

### 消息中心
```
1. 点击右上角铃铛图标
2. 查看未读消息
3. 点击"全部已读"标记
```

### 快捷操作
```
1. 点击"快速创建"按钮
2. 选择操作类型
3. 填写信息并提交
```

### 批量操作
```
1. 勾选多条记录
2. 显示批量操作栏
3. 选择分配/导出/删除
```

### 数据导出
```
1. 点击"导出"按钮
2. 选择格式和字段
3. 下载文件
```

### 导入数据
```
1. 下载导入模板
2. 填写 Excel 数据
3. 上传文件验证
4. 确认导入
```

### 回收站
```
1. 访问回收站页面
2. 查看已删除记录
3. 点击恢复或永久删除
```
