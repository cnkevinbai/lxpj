## 🌍 国际化 (i18n)

### 支持语言

| 语言 | 代码 | 状态 |
|-----|------|------|
| 简体中文 | zh-CN | ✅ 默认 |
| 英语 (美国) | en-US | ✅ |
| 英语 (英国) | en-GB | ✅ |
| 西班牙语 | es-ES | ⏳ 二期 |
| 法语 | fr-FR | ⏳ 二期 |
| 阿拉伯语 | ar-SA | ⏳ 二期 (RTL) |

---

### 实现方案

**技术栈**: `next-i18next`

**文件结构**:
```
/locales/
├── zh-CN/
│   ├── common.json      # 通用文案
│   ├── navigation.json  # 导航菜单
│   ├── product.json     # 产品相关
│   └── footer.json      # 页脚
├── en-US/
│   ├── common.json
│   ├── navigation.json
│   ├── product.json
│   └── footer.json
```

**使用示例**:
```jsx
// 组件中使用
import { useTranslation } from 'next-i18next'

function HeroBanner() {
  const { t } = useTranslation('common')
  return <h1>{t('hero.title')}</h1>
}
```

### 组件设计原则

1. **单一职责**: 每个组件只做一件事

## 🎬 动效规范

### 原则

- **克制**: 不滥用动效，只在必要时使用
- **流畅**: 使用缓动函数，避免生硬
- **有意义**: 动效服务于功能，不只是装饰

---

### 缓动函数

```css
/* 标准缓动 */
--ease-out: cubic-bezier(0.215, 0.61, 0.355, 1);
--ease-in-out: cubic-bezier(0.645, 0.045, 0.355, 1);
--ease-in: cubic-bezier(0.55, 0.055, 0.675, 0.19);
--ease-out: cubic-bezier(0.215, 0.61, 0.355, 1);
```

---

### 动效场景

| 场景 | 时长 | 缓动 | 说明 |
|-----|------|------|------|
| Hover 反馈 | 150ms | ease-out | 按钮、卡片 |
| 页面切换 | 300ms | ease-in-out | 路由切换 |
| 模态框 | 200ms | ease-out | 弹出、关闭 |
| 轮播切换 | 500ms | ease-in-out | Banner 轮播 |
| 加载动画 | 持续 | linear | Loading |
