## ⚙️ 配置说明

### 环境变量（.env）

```bash
# 网站统计
BAIDU_TONGJI=你的百度统计 ID
GA_ID=你的 Google Analytics ID

# 在线客服
CHAT_PROVIDER=meiqia
CHAT_ID=你的客服系统 ID

# SEO
SITE_URL=https://www.evcart.com
```

### 客服系统配置

**美洽**（推荐）:
1. 注册 https://www.meiqia.com
2. 创建企业，获取企业 ID
3. 配置 `CHAT_PROVIDER=meiqia` 和 `CHAT_ID`

**环信**:
1. 注册 https://www.easemob.com
2. 创建应用，获取应用 ID
3. 配置 `CHAT_PROVIDER=easemob`

**腾讯企点**:
1. 注册 https://qidian.qq.com
2. 获取站点 ID
3. 配置 `CHAT_PROVIDER=qidian`

## 🚀 部署步骤

```bash
# 1. 安装依赖
cd website
npm install

# 2. 配置环境变量
cp .env.example .env
# 编辑 .env 填入配置

# 3. 本地测试
npm run dev
# 访问 http://localhost:3000

# 4. 构建检查
npm run build

# 5. 提交代码
git add .
git commit -m "feat: 官网 v2.0 升级 - 新增案例/解决方案/新闻/视频板块"
git push

# 6. 生产部署（Docker）
cd ..
docker-compose restart website

# 7. 验证
# 访问所有新页面，检查功能正常
```
