## 🔗 质量数据互通完整链路

### 1. 质量 ↔ 采购互通 ✅

**互通 API**:
```typescript
// 来料检验合格自动入库
POST /api/v1/quality/integration/inspections/:id/stock-in

// 获取供应商质量统计

// 更新供应商质量评分
```

**互通流程**:
```

  待检区    检验结果   库存增加    质量档案
```

**数据同步**:
- ✅ 检验合格自动入库
