## 📋 外贸订单管理

### 订单流程

```
待确认 → 定金已到 → 生产中 → 备货完成 → 已发货 → 已完成
```

### 订单字段

| 字段 | 说明 |
|-----|------|
| 订单编号 | 自动生成 |
| PI 编号 | 形式发票号 |
| CI 编号 | 商业发票号 |
| 产品详情 | 产品、数量、单价 |
| 金额信息 | 总金额、货币、已付金额 |
| 贸易信息 | 贸易术语、装运港、目的港 |
| 付款信息 | 付款方式、定金比例 |
| 物流信息 | 发货时间、物流单号、单据 |

### 外贸单据

| 单据 | 说明 |
|-----|------|
| PI | 形式发票 (Proforma Invoice) |
| CI | 商业发票 (Commercial Invoice) |
| PL | 装箱单 (Packing List) |
| BL | 提单 (Bill of Lading) |
| CO | 原产地证 (Certificate of Origin) |

## 🔌 API 接口

### 外贸线索

| 接口 | 方法 | 说明 |
|-----|------|------|
| `/foreign-leads` | POST | 创建外贸线索 |
| `/foreign-leads` | GET | 获取外贸线索列表 |
| `/foreign-leads/:id` | GET | 获取线索详情 |
| `/foreign-leads/:id` | PUT | 更新线索 |
| `/foreign-leads/:id` | DELETE | 删除线索 |
| `/foreign-leads/stats` | GET | 获取线索统计 |
