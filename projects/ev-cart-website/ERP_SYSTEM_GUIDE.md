## 📊 系统概览

道达智能 ERP 系统是企业资源计划核心，涵盖 10 大功能模块，实现从采购到生产、从库存到财务的全流程管理。

### 核心能力
- ✅ 生产管理 (MRP + 工单 + 进度跟踪)
- ✅ 采购管理 (供应商 + 采购订单 + 对账)
- ✅ 库存管理 (多仓库 + 调拨 + 盘点)
- ✅ 财务管理 (应收应付 + 成本核算)
- ✅ 质量管理 (质检流程 + 不合格品处理)
- ✅ 设备管理 (全生命周期 + 保养计划)
- ✅ 成本管理 (标准成本 + 实际成本)
- ✅ 资产管理 (固定资产 + 折旧)
- ✅ 价格管理 (多层级价格体系)
- ✅ 出口管理 (外贸订单 + 报关)

## 📦 模块详解

### 1. 生产管理 (Production Management)

**路径**: `/production-orders`  
**后端**: `backend/src/modules/erp-production/`

#### 核心功能
| 功能 | 说明 | 状态 |
|------|------|------|
| 生产订单 | 创建、编辑、下达生产工单 | ✅ |
| MRP 运算 | 物料需求计划自动计算 | ✅ |
| 生产进度 | 实时跟踪生产进度 | ✅ |
| 工序管理 | 定义生产工艺路线 | ✅ |
| 报工管理 | 工序报工、工时统计 | ✅ |
| 生产看板 | 可视化生产状态 | ✅ |

#### 业务流程
```
销售订单 → MRP 运算 → 采购申请 + 生产工单 → 领料 → 生产 → 质检 → 入库
```

#### 关键数据表
- `erp_production_orders` - 生产订单主表
- `erp_production_operations` - 工序记录
- `erp_material_requirements` - 物料需求
- `erp_work_reports` - 报工记录

| 设备健康度 | 健康评估、预测性维护 | ✅ |

#### OEE 计算
```
OEE = 可用性 × 性能 × 质量
可用性 = 运行时间 / 计划时间
性能 = 实际产量 / 理论产量
质量 = 合格品数量 / 总产量
```

## 🔄 核心业务流程

### 采购到付款 (P2P)
```
需求申请 → 采购订单 → 供应商发货 → 质检 → 入库 → 对账 → 付款
```

### 生产到入库 (M2S)
```
销售订单 → MRP 运算 → 生产工单 → 领料 → 生产 → 质检 → 成品入库
```

### 订单到收款 (O2C)
```
销售订单 → 信用审核 → 生产/备货 → 发货 → 开票 → 收款
```

## 📈 关键指标 (KPI)

| 指标 | 公式 | 目标值 |
|------|------|--------|
| 订单交付率 | 按时交付订单数 / 总订单数 | ≥ 95% |
| 库存周转率 | 销售成本 / 平均库存 | ≥ 6 次/年 |
| 采购及时率 | 及时到货批次 / 总采购批次 | ≥ 90% |
| 产品合格率 | 合格品数量 / 总产量 | ≥ 98% |
| 设备 OEE | 可用性×性能×质量 | ≥ 85% |
| 成本偏差率 | (实际成本 - 标准成本) / 标准成本 | ≤ 5% |

### 后端技术栈

- **ORM**: Prisma / TypeORM

### 前端技术栈
- **框架**: React 18 + Vite
- **UI 库**: Ant Design 5
- **图表**: @ant-design/plots
- **状态**: React Hooks
