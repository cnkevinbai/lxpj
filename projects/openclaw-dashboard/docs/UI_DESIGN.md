# OpenClaw 可视化操作面板 - UI 设计规范

> **版本**: v1.0
> **设计日期**: 2026-03-18
> **设计师**: Maya (ui-ux-designer)

---

## 一、设计原则

- **现代专业** - 简洁有力的视觉风格
- **暗色主题** - 护眼、沉浸式体验
- **响应式** - 适配桌面、平板、手机
- **一致性** - 统一的组件和交互

---

## 二、配色方案

### 2.1 主色调

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#3B82F6` | 蓝色 - 按钮、链接、强调 |
| 成功 | `#10B981` | 绿色 - 成功状态 |
| 警告 | `#F59E0B` | 橙色 - 警告提示 |
| 危险 | `#EF4444` | 红色 - 错误、删除 |

### 2.2 背景色

| 用途 | 色值 | 说明 |
|------|------|------|
| 深背景 | `#0F172A` | 页面背景 |
| 卡片背景 | `#1E293B` | 卡片、侧边栏 |
| 悬停背景 | `#334155` | hover 状态 |
| 边框 | `#475569` | 分割线 |

### 2.3 文字色

| 用途 | 色值 | 说明 |
|------|------|------|
| 主文字 | `#F9FAFB` | 标题、正文 |
| 次文字 | `#94A3B8` | 说明、描述 |
| 禁用 | `#64748B` | 禁用状态 |

## 三、布局设计

### 3.1 整体布局

```
┌────────────────────────────────────────────────────────┐
│  Header (56px)                          [用户] [设置] │
├──────────┬─────────────────────────────────────────────┤
│          │                                             │
│  Sidebar │              Main Content                  │
│  (240px) │                                             │
│          │                                             │
│  导航菜单 │              (min-height: calc(100vh - 56px) │
│          │                                             │
│          │                                             │
├──────────┴─────────────────────────────────────────────┤
│  Footer (可选)                                        │
└────────────────────────────────────────────────────────┘
```

### 3.2 响应式断点

| 设备 | 断点 | 侧边栏 |
|------|------|---------|
| Desktop | ≥1024px | 固定 240px |
| Tablet | 640-1024px | 收起 64px |
| Mobile | <640px | 隐藏，点击展开 |

## 五、组件规范

### 5.1 按钮

| 类型 | 样式 | 使用场景 |
|------|------|----------|
| Primary | 蓝色填充 | 主要操作 |
| Secondary | 边框样式 | 次要操作 |
| Text | 纯文字 | 链接式按钮 |
| Danger | 红色填充 | 危险操作 |

### 5.2 状态徽章

| 状态 | 颜色 |
|------|------|
| 在线 | `#10B981` 绿色 |
| 离线 | `#64748B` 灰色 |
| 处理中 | `#3B82F6` 蓝色 |
| 警告 | `#F59E0B` 橙色 |
| 错误 | `#EF4444` 红色 |

### 5.3 任务优先级

| 优先级 | 颜色 | 图标 |
|--------|------|------|
| Critical | 红 | 🔴 |
| High | 橙 | 🟠 |
| Medium | 黄 | 🟡 |
| Low | 灰 | ⚪ |

## 六、交互设计

### 6.1 动画时长

| 类型 | 时长 |
|------|------|
| 微交互 | 100ms |
| 常规动画 | 200ms |
| 页面切换 | 300ms |

### 6.2 快捷键

| 快捷键 | 功能 |
|--------|------|
| Enter | 发送消息 |
| Ctrl+K | 搜索 |
| Ctrl+N | 新建会话 |
| Esc | 关闭弹窗 |

### 6.3 危险操作确认

```
用户点击危险按钮
    ↓
弹出确认对话框
    ↓
输入操作原因（可选）
    ↓
确认执行
```
