# 智能代理路由 + 任务窗口多开系统

## 功能概述

主 Agent (渔晓白) 能够自动识别任务关键词，将任务分配给专业 Agent，并自动创建独立的任务窗口，支持窗口驻留、挂起、关闭和多窗口展示。

### 2. 任务窗口功能

## 使用示例

### 1. 自动路由 + 窗口创建

用户输入：
```
帮我设计一个微服务架构
```

系统响应：
1. 检测关键词 "微服务"、"架构"
2. 自动切换到 Morgan (架构师)
3. 创建独立任务窗口
4. 提交后台任务到队列
5. 流式返回响应

### 2. 手动创建窗口

1. 点击任务栏的 "+ 新建任务窗口"
2. 选择代理（如 Ryan）
3. 打开独立窗口

### 3. 窗口操作

| 操作 | 快捷方式 |
|------|----------|
| 驻留窗口 | 点击 📌 |
| 最小化 | 点击 ➖ |
| 关闭 | 点击 ✕ |
| 切换窗口 | 任务栏点击 |

## 技术特性

### 多窗口隔离

- 每个窗口独立的 Zustand Store
- 独立的 WebSocket 连接
- 独立的会话上下文
- 窗口间通过 BroadcastChannel 同步

### 7x24 高可用

- 任务持久化 (localStorage)
- 失败自动重试
- 窗口状态保存
- 离线任务缓存

### 并发控制

- 最大并发 3 个任务
- Agent 独占保护
- 优先级队列
