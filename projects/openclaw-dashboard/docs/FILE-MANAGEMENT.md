# 文件管理模块文档

## 功能概述

OpenClaw 控制面板的文件管理模块提供了一个完整的文件管理系统，支持以下功能：

### 后端 API (NestJS)

#### 端点

- `GET /api/files` - 获取所有文件列表
- `GET /api/files/:id` - 获取指定文件的信息
- `POST /api/files/upload` - 上传文件
- `DELETE /api/files/:id` - 删除文件
- `GET /api/files/:id/download` - 下载文件
- `GET /api/files/:id/content` - 获取文件内容（用于预览）

#### 文件服务特性

- **文件存储**：文件存储在 `uploads/` 目录下
- **文件类型识别**：自动识别文件类型并返回 MIME 类型
- **文件大小显示**：自动格式化文件大小（B, KB, MB, GB, TB）
- **文件预览**：支持图片和文本文件的预览

### 前端页面 (React + TypeScript)

#### 功能

- **文件列表视图**：支持列表和网格两种视图模式
- **文件上传**：支持拖拽上传和点击上传
- **文件搜索**：实时搜索文件名
- **文件筛选**：按文件名筛选
- **分页显示**：每页显示最多 50 个文件
- **文件预览**：
  - 图片文件：直接显示图片
  - 文本文件：显示文件内容预览
- **文件操作**：
  - 下载：下载文件到本地
  - 删除：删除文件
  - 预览：查看文件内容

#### UI 设计

- 现代化深色主题
- 仪表盘科技风格
- 响应式布局
- 优雅的过渡动画

## 技术栈

- **后端**：NestJS + TypeScript
- **前端**：React 18 + TypeScript + Vite
- **样式**：Tailwind CSS
- **HTTP 客户端**：axios

## 使用说明

### 后端配置

1. 确保 `uploads/` 目录存在且有写入权限
2. 文件将自动存储在 `uploads/` 目录下
3. API 端点已注册到 `ApiModule`

### 前端使用

1. 访问 `/files` 路由进入文件管理页面
2. 点击"选择文件"或拖拽文件到上传区域
3. 使用搜索栏快速查找文件
4. 点击文件名进行预览
5. 使用下载按钮保存文件到本地
6. 点击删除按钮删除不需要的文件

## 注意事项

- 文件大小限制：建议上传文件不超过 10MB
- 文件类型：支持所有常见文件类型
- 预览限制：文本文件只预览前 500 个字符
- 安全性：文件路径经过验证，防止路径遍历攻击
