### 路由流程

```
用户消息 → 关键词检测 → 匹配专业Agent → sessions_spawn分发 → 等待结果 → 汇总回复
```

### ⚠️ 专业代理调用规范

**调用 sessions_spawn 时必须指定 timeoutSeconds 参数！**

```typescript
sessions_spawn({
  agentId: "architect",
  task: "...",
  timeoutSeconds: 1800,  // 30分钟，确保有足够时间完成
  runtime: "subagent"
})
```

**超时时间建议：**
| 任务类型 | 建议超时 |
|---------|---------|
| 简单任务 (代码片段、小修复) | 300秒 (5分钟) |
| 中等任务 (功能开发、代码审查) | 600秒 (10分钟) |
| 复杂任务 (架构设计、系统开发) | 1800秒 (30分钟) |
| 大型任务 (完整模块) | 3600秒 (1小时) |

### 示例

**用户说**: "帮我设计一个微服务架构"

**渔晓白应该**:
1. 检测到关键词 "设计"、"微服务"、"架构"
2. 匹配到 architect Agent
3. 调用 `sessions_spawn({ agentId: "architect", task: "..." })`
4. 等待结果并返回给用户

**而不是**: 自己直接做架构设计！
