## 三、验收标准

### Phase 1 验收

- [ ] API 路径统一为 `/api/v1/*`
- [ ] 错误响应包含标准错误码
- [ ] ESLint 检查通过
- [ ] 核心表有 ext_data 字段

### Phase 2 验收

- [ ] 可通过配置禁用模块
- [ ] 禁用的模块菜单不显示
- [ ] 禁用的模块 API 返回 403

### Phase 3 验收

- [ ] 支持创建多个租户
- [ ] 不同租户数据隔离
- [ ] 用户可切换租户

### Phase 4 验收

- [ ] 支持 Webhook 配置
- [ ] 事件可触发 Webhook
- [ ] 支持 API Key 认证

### Phase 5 验收

- [ ] 支持运行时加载模块
- [ ] 支持运行时卸载模块
- [ ] 支持插件安装/卸载
