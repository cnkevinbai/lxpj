## 一、项目概述

### 1.1 项目定位

**道达智能数字化平台** 是一个面向企业的综合数字化管理平台，包含：

- **对外官网** (Website) - 品牌门户、产品展示、解决方案
- **对内门户** (Portal) - CRM、ERP、财务、人事、售后服务
- **后端服务** (Backend) - RESTful API、业务逻辑、数据管理

### 1.2 核心价值

```
对外 → 品牌展示、产品推广、客户触达
对内 → 业务管理、流程协同、数据分析
平台 → 多租户、模块化、可扩展
```
