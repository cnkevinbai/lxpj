## 📝 开发规范

### 模块结构规范

```
module-name/
├── module.yaml           # 模块配置文件
├── pom.xml              # Maven依赖配置
├── src/
│   ├── main/
│   │   ├── java/
│   │   │   └── com/daod/iov/modules/modulename/
│   │   │       ├── ModuleNameService.java    # 主服务类
│   │   │       ├── entity/                   # 实体类
│   │   │       ├── dto/                      # 数据传输对象
│   │   │       ├── service/                  # 业务服务
│   │   │       ├── event/                    # 事件类
│   │   │       └── exception/                # 异常类
│   │   └── resources/
│   │       └── application.yml               # 配置文件
│   └── test/
│       └── java/
│           └── ModuleNameServiceTest.java    # 单元测试
```

### 模块接口规范

所有模块必须实现 `IModule` 接口：

```java
public interface IModule {
    ModuleMetadata getMetadata();      // 获取模块元数据
    void initialize(ModuleContext context);  // 初始化
    void start();                      // 启动
    void stop();                       // 停止
    void destroy();                    // 销毁
    ModuleState getState();            // 获取状态
    HealthStatus getHealthStatus();    // 获取健康状态
}
```
