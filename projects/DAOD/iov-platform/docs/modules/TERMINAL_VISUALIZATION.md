**项目**: 道达智能车辆管理平台 (iov-platform)  
**版本**: 1.0.0  
**日期**: 2026-03-24

## 1. 方案概述

### 1.1 目标

实现 T-BOX 终端设备的可视化接入管理，包括：
- 终端连接状态实时监控
- 终端设备在线地图
- 接入日志与告警
- 远程指令下发
- 终端配置管理

## 3. 技术实现

### 3.1 前端技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| React | 18.x | UI框架 |
| TypeScript | 5.x | 类型安全 |
| Ant Design | 5.x | UI组件库 |
| Leaflet | 1.9.x | 地图展示 |
| WebSocket | - | 实时通信 |
| Axios | 1.x | HTTP请求 |
| Vite | 5.x | 构建工具 |

### 3.2 实时通信架构

```
┌─────────────┐     WebSocket      ┌─────────────┐     MQTT      ┌─────────────┐
│   前端      │ ◀────────────────▶ │ API Gateway │ ◀───────────▶ │   EMQX      │
│  iov-portal │                    │  /ws/terminals│              │  Broker     │
└─────────────┘                    └─────────────┘               └─────────────┘
                                          │
                                          ▼
                                   ┌─────────────┐
                                   │  Redis      │
                                   │  Pub/Sub    │
                                   └─────────────┘
```

### 3.3 API 接口设计

#### 终端管理 API

| 方法 | 路径 | 描述 |
|------|------|------|
| GET | /api/terminals | 终端列表 |
| GET | /api/terminals/{id} | 终端详情 |
| GET | /api/terminals/{id}/location | 终端位置 |
| GET | /api/terminals/{id}/logs | 通信日志 |
| POST | /api/terminals/{id}/command | 下发指令 |
| GET | /api/terminals/stats | 统计数据 |

#### WebSocket 主题

| 主题 | 描述 |
|------|------|
| /ws/terminals/online | 终端上线通知 |
| /ws/terminals/offline | 终端离线通知 |
| /ws/terminals/location | 位置更新 |
| /ws/terminals/alarm | 告警通知 |
