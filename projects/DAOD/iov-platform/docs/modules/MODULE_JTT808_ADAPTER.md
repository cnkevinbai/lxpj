## 1. 模块概述

JT/T 808协议适配器实现了道路运输车辆卫星定位系统终端通讯协议，是车联网平台与车载终端通信的核心组件。

### 1.1 协议版本支持

| 版本 | 说明 |
|------|------|
| 2011版 | JT/T 808-2011 |
| 2013版 | JT/T 808-2013 |
| 2019版 | JT/T 808-2019 (推荐) |

### 1.2 核心功能

| 功能 | 消息ID | 说明 |
|------|--------|------|
| 终端注册 | 0x0100 | 终端首次接入平台 |
| 终端鉴权 | 0x0102 | 身份认证 |
| 位置上报 | 0x0200 | GPS定位数据上报 |
| 心跳检测 | 0x0002 | 保持在线状态 |
| 批量位置 | 0x0704 | 批量位置数据上传 |
| 多媒体上传 | 0x0800/0x0801 | 图片/视频上传 |

## 3. 消息协议

### 3.1 消息结构

```
┌──────────────────────────────────────────────────────────────┐
│                       消息头 (Header)                         │
├──────┬──────┬──────────┬─────────────┬──────────┬───────────┤
│ 标识位│消息ID│消息体属性│ 终端手机号   │ 流水号   │ 消息包封装 │
│ 0x7E │2字节 │  2字节   │   6字节     │  2字节   │  (可选)   │
└──────┴──────┴──────────┴─────────────┴──────────┴───────────┘
┌──────────────────────────────────────────────────────────────┐
│                       消息体 (Body)                           │
├──────────────────────────────────────────────────────────────┤
│                     业务数据 (变长)                            │
└──────────────────────────────────────────────────────────────┘
┌──────────────────────────────────────────────────────────────┐
│                       校验码 (Checksum)                       │
├──────────────────────────────────────────────────────────────┤
│                     CRC校验 (1字节)                           │
└──────────────────────────────────────────────────────────────┘
┌──────────────────────────────────────────────────────────────┐
│                       标识位                                  │
├──────────────────────────────────────────────────────────────┤
│                        0x7E                                   │
└──────────────────────────────────────────────────────────────┘
```

## 4. API设计

### 4.1 核心接口

```java
public class JTT808Adapter implements IModule {
    
    /**
     * 处理接收消息
     */
    public void handleMessage(ChannelHandlerContext ctx, JTT808Message message);
    
    /**
     * 注册消息处理器
     */
    public void registerHandler(int messageId, MessageHandler handler);
    
    /**
     * 添加终端会话
     */
    public void addTerminalSession(String phone, TerminalSession session);
    
    /**
     * 移除终端会话
     */
    public void removeTerminalSession(String phone);
    
    /**
     * 获取终端会话
     */
    public TerminalSession getTerminalSession(String phone);
    
    /**
     * 获取在线终端数
     */
    public int getOnlineCount();
    
    /**
     * 下发文本消息
     */
    public void sendTextMessage(String phone, String text, int urgency);
    
    /**
     * 下发终端参数
     */
    public void sendTerminalParameters(String phone, Map<Integer, Object> params);
}
```

### 4.2 消息处理器接口

```java
public interface MessageHandler {
    JTT808Message handle(JTT808Message message, ChannelHandlerContext ctx);
}
```

## 5. 配置项

```yaml
jtt808:
  serverPort: 8888              # TCP服务端口
  maxConnections: 5000          # 最大连接数
  idleTimeout: 300000           # 空闲超时(ms)
  heartbeatInterval: 60000      # 心跳间隔(ms)
  version: "2019"               # 协议版本
  encoding: "GBK"               # 消息编码
  enableEncryption: false       # 是否启用加密
```

## 6. 下行指令

### 6.1 支持的下行指令

| 消息ID | 名称 | 说明 |
|--------|------|------|
| 0x8001 | 平台通用应答 | 对上行消息的应答 |
| 0x8100 | 终端注册应答 | 注册成功/失败 |
| 0x8103 | 终端参数设置 | 设置终端参数 |
| 0x8300 | 文本信息下发 | 下发文本消息 |
| 0x8500 | 车辆控制 | 锁车/解锁等 |
| 0x8600 | 设置圆形区域 | 电子围栏 |
| 0x8800 | 多媒体检索 | 请求多媒体数据 |

### 6.2 下发示例

```java
// 下发文本消息
adapter.sendTextMessage("13800138000", "请到指定地点充电", 1);

// 下发终端参数
Map<Integer, Object> params = new HashMap<>();
params.put(0x0025, 60);  // 心跳间隔60秒
params.put(0x0026, 3);   // TCP重连次数
adapter.sendTerminalParameters("13800138000", params);
```
