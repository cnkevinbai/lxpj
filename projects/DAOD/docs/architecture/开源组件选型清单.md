## 一、消息中间件

### 1.1 EMQX (MQTT Broker)

| 项目 | 说明 |
|-----|------|
| **官网** | https://www.emqx.io/ |
| **版本** | 5.0+ |
| **许可证** | Apache 2.0 |
| **选型理由** | 国产开源，支持国密算法，百万级并发连接，内置规则引擎 |

**核心特性:**
- 支持MQTT 5.0/3.1.1协议
- 内置规则引擎，支持SQL语法
- 支持集群部署，线性扩展
- 提供Dashboard管理界面
- 支持JT/T 808、GB/T 32960协议插件
- 国密SM2/SM3/SM4算法支持

**部署配置:**
```yaml
# docker-compose.yml
version: '3'
services:
  emqx:
    image: emqx/emqx:5.0
    container_name: emqx
    environment:
      - EMQX_NAME=emqx
      - EMQX_HOST=0.0.0.0
      - EMQX_DASHBOARD__DEFAULT_USERNAME=admin
      - EMQX_DASHBOARD__DEFAULT_PASSWORD=public
    ports:
      - "1883:1883"    # MQTT
      - "8083:8083"    # WebSocket
      - "8084:8084"    # WSS
      - "8883:8883"    # MQTTS
      - "18083:18083"  # Dashboard
    volumes:
      - ./emqx/data:/opt/emqx/data
      - ./emqx/log:/opt/emqx/log
```
